# 1.7 FastAPI 快速入门

## 1 技术简介

FastAPI 是一个现代、高性能的 Python Web 框架，专为构建 API 而生。它基于 Starlette（高性能异步框架）和 Pydantic（数据验证库），具有以下核心特点：

- **高性能**：性能媲美 Node.js 和 Go，是 Python 最快的 Web 框架之一
- **快速开发**：开发效率提升 200%-300%，代码量减少 40%
- **类型安全**：基于 Python 类型提示，IDE 支持完善，自动补全和错误检查
- **自动文档**：自动生成 OpenAPI (Swagger) 和 ReDoc 文档
- **异步原生**：原生支持 `async`/`await`，适合高并发场景

**对比 Flask/Django**：
| 特性 | FastAPI | Flask | Django |
|------|---------|-------|--------|
| 性能 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐ |
| 异步支持 | 原生 | 需扩展 | 需扩展 |
| 自动文档 | ✅ | ❌ | ❌ |
| 数据验证 | 内置 | 需手动 | 需手动 |
| 学习曲线 | 中等 | 低 | 高 |

## 2 技术背景

### 为什么需要 FastAPI？

**传统 Python Web 开发的痛点**：

1. **Flask**：轻量但功能单一，需自行集成验证、文档、异步等
2. **Django**：功能齐全但过于笨重，不适合微服务和 API 开发
3. **性能瓶颈**：WSGI 同步模型难以应对高并发

**FastAPI 的解决方案**：

- 基于 ASGI（异步服务器网关接口），原生支持异步
- 内置数据验证（Pydantic），无需额外库
- 自动生成 API 文档，无需手动维护
- 类型提示驱动，开发体验接近 TypeScript

### 与 Go 的对比（给 Go 开发者）

| 概念 | Go (Gin) | FastAPI |
|------|----------|---------|
| 路由定义 | `router.GET("/path", handler)` | `@app.get("/path")` |
| 参数绑定 | 手动解析 | 自动解析（类型驱动） |
| 数据验证 | 手动验证 | Pydantic 自动验证 |
| 中间件 | `router.Use()` | `@app.middleware()` |
| 并发模型 | Goroutine | `async`/`await` |

**关键差异**：
- Go 是编译型语言，Python 是解释型
- FastAPI 的类型提示是可选的（但强烈推荐），Go 是强类型
- FastAPI 的自动文档是开箱即用的，Go 需要额外工具（如 swaggo）

## 3 核心概念

### 3.1 路径操作（Path Operations）

类似于 Go 的 HTTP 路由处理函数，FastAPI 使用装饰器定义路由：

```python
# Go 风格（对比）
# router.GET("/users/{id}", getUser)

# FastAPI 风格
@app.get("/users/{user_id}")
def get_user(user_id: int):
    return {"user_id": user_id}
```

**支持的 HTTP 方法**：
- `@app.get()` - 获取资源
- `@app.post()` - 创建资源
- `@app.put()` - 更新资源（完整替换）
- `@app.patch()` - 更新资源（部分更新）
- `@app.delete()` - 删除资源

### 3.2 Pydantic 模型

用于定义数据结构，自动进行验证和序列化：

```python
from pydantic import BaseModel

class User(BaseModel):
    """用户数据模型"""
    id: int
    name: str
    email: str
    is_active: bool = True  # 默认值
```

**与 Go struct 对比**：
```go
// Go
 type User struct {
     ID       int    `json:"id"`
     Name     string `json:"name"`
     Email    string `json:"email"`
     IsActive bool   `json:"is_active"`
 }
```

### 3.3 依赖注入（Dependency Injection）

FastAPI 提供了强大的依赖注入系统，类似于 Go 的接口和依赖管理：

```python
from fastapi import Depends

# 定义依赖
async def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

# 使用依赖
@app.get("/users")
def read_users(db: Session = Depends(get_db)):
    return db.query(User).all()
```

### 3.4 异步支持

FastAPI 原生支持异步，使用 `async`/`await` 语法：

```python
import asyncio

@app.get("/slow-endpoint")
async def slow_endpoint():
    """模拟耗时操作"""
    await asyncio.sleep(1)  # 非阻塞等待
    return {"message": "Done"}
```

**注意**：只有调用异步库（如 `asyncio`、`httpx`、`aiomysql`）时才需要 `async`，纯 CPU 操作不需要。

## 4 工作原理

### 4.1 请求处理流程

```
Request
    ↓
Middleware（中间件处理）
    ↓
Dependency（依赖注入）
    ↓
Path Operation Function（路由处理函数）
    ↓
Pydantic Validation（数据验证）
    ↓
Business Logic（业务逻辑）
    ↓
Response Model（响应模型序列化）
    ↓
Response
```

### 4.2 ASGI 服务器

FastAPI 基于 ASGI（Asynchronous Server Gateway Interface）标准，默认使用 Uvicorn 服务器：

```
┌─────────────────────────────────────┐
│           Uvicorn Server            │
│  ┌─────────────────────────────┐   │
│  │      ASGI Application       │   │
│  │      (FastAPI Instance)     │   │
│  │  ┌─────────────────────┐   │   │
│  │  │   Starlette         │   │   │
│  │  │   (HTTP + WebSocket)│   │   │
│  │  └─────────────────────┘   │   │
│  │  ┌─────────────────────┐   │   │
│  │  │   Pydantic          │   │   │
│  │  │   (Validation)      │   │   │
│  │  └─────────────────────┘   │   │
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

### 4.3 自动文档生成

FastAPI 自动读取类型提示和 docstring，生成 OpenAPI 规范：

```python
@app.get("/items/{item_id}")
def read_item(item_id: int, q: str = None):
    """
    获取商品详情
    
    - **item_id**: 商品 ID
    - **q**: 可选查询参数
    """
    return {"item_id": item_id, "q": q}
```

访问 `/docs` 可查看 Swagger UI 文档。

### 5.2 请求参数类型

```python
from fastapi import FastAPI, Query, Path, Body
from pydantic import BaseModel
from typing import Optional, List

app = FastAPI()

# 1. 路径参数
@app.get("/users/{user_id}")
def get_user(user_id: int = Path(..., gt=0, description="用户ID")):
    """路径参数自动转换类型"""
    return {"user_id": user_id}

# 2. 查询参数
@app.get("/search")
def search(
    keyword: str = Query(..., min_length=2, description="搜索关键词"),
    page: int = Query(1, ge=1, description="页码"),
    size: int = Query(20, ge=1, le=100, description="每页数量")
):
    """查询参数自动验证"""
    return {"keyword": keyword, "page": page, "size": size}

# 3. 请求体（POST）
class CreateUserRequest(BaseModel):
    """创建用户请求"""
    username: str
    email: str
    age: Optional[int] = Field(None, ge=0, le=150)

@app.post("/users")
def create_user(user: CreateUserRequest):
    """请求体自动解析和验证"""
    return {"user": user}

# 4. 混合参数
@app.put("/users/{user_id}")
def update_user(
    user_id: int = Path(..., description="用户ID"),
    user_data: CreateUserRequest = Body(...),
    notify: bool = Query(True, description="是否发送通知")
):
    """同时使用路径参数、请求体和查询参数"""
    return {
        "user_id": user_id,
        "data": user_data,
        "notify": notify
    }
```

### 5.5 中间件

```python
from fastapi import FastAPI, Request
import time

app = FastAPI()

# 注册中间件
@app.middleware("http")
async def add_process_time_header(request: Request, call_next):
    """
    添加处理时间头的中间件
    
    类似于 Go 的 gin.Use(middleware)
    """
    start_time = time.time()
    
    # 调用下一个中间件或路由处理函数
    response = await call_next(request)
    
    process_time = time.time() - start_time
    response.headers["X-Process-Time"] = str(process_time)
    
    return response

# CORS 中间件
from fastapi.middleware.cors import CORSMiddleware

    CORSMiddleware,
    allow_origins=["*"],  # 生产环境应指定具体域名
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

## 6 Python代码实现与逐行解析

### 6.1 完整用户管理 API

```python
"""
用户管理 API 示例
演示 FastAPI 的核心功能：路由、参数验证、依赖注入、数据库操作
"""

from fastapi import FastAPI, HTTPException, Depends, status
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from pydantic import BaseModel, Field, EmailStr
from typing import List, Optional
from datetime import datetime
import uvicorn

# ========================================
# 1. 数据模型定义（Pydantic）
# ========================================

class UserBase(BaseModel):
    """用户基础模型（共享字段）"""
    username: str = Field(..., min_length=3, max_length=50, description="用户名")
    email: EmailStr = Field(..., description="邮箱地址")
    is_active: bool = Field(default=True, description="是否激活")

class UserCreate(UserBase):
    """创建用户请求模型"""
    password: str = Field(..., min_length=6, description="密码")

class UserUpdate(BaseModel):
    """更新用户请求模型（所有字段可选）"""
    username: Optional[str] = Field(None, min_length=3, max_length=50)
    email: Optional[EmailStr] = None
    is_active: Optional[bool] = None

class UserResponse(UserBase):
    """用户响应模型（返回给客户端）"""
    id: int = Field(..., description="用户ID")
    created_at: datetime = Field(default_factory=datetime.now, description="创建时间")
    
    class Config:
        # 启用 ORM 模式，支持从 SQLAlchemy 模型直接转换
        from_attributes = True

class UserInDB(UserResponse):
    """数据库用户模型（包含敏感信息）"""
    hashed_password: str

# ========================================
# 2. 模拟数据库（实际项目应使用真实数据库）
# ========================================

# 内存数据库（仅用于演示）
fake_users_db = {
    1: UserInDB(
        id=1,
        username="admin",
        email="admin@example.com",
        hashed_password="hashed_admin_password",
        is_active=True,
        created_at=datetime.now()
    ),
    2: UserInDB(
        id=2,
        username="user1",
        email="user1@example.com",
        hashed_password="hashed_user1_password",
        is_active=True,
        created_at=datetime.now()
    )
}

# ID 计数器
user_id_counter = 3

# ========================================
# 3. 依赖函数
# ========================================

security = HTTPBearer()

def get_current_user(credentials: HTTPAuthorizationCredentials = Depends(security)):
    """
    获取当前登录用户（模拟认证）
    
    Args:
        credentials: HTTP Bearer Token
    
    Returns:
        UserResponse: 当前用户信息
    
    Raises:
        HTTPException: 认证失败时抛出 401 错误
    """
    token = credentials.credentials
    # 实际项目中应验证 JWT Token
    if token != "valid_token":
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="无效的认证凭据",
            headers={"WWW-Authenticate": "Bearer"}
        )
    # 返回第一个用户作为示例
    return fake_users_db[1]

def get_db():
    """
    获取数据库连接（模拟依赖）
    
    Yields:
        dict: 模拟数据库连接
    """
    # 实际项目中应创建和关闭真实的数据库连接
    db = {"connection": "active"}
    try:
        yield db
    finally:
        db["connection"] = "closed"

# ========================================
# 4. 创建 FastAPI 应用
# ========================================

app = FastAPI(
    title="用户管理系统 API",
    description="演示 FastAPI 的用户管理 API，包含完整的 CRUD 操作",
    version="1.0.0",
    docs_url="/docs",      # Swagger UI 文档地址
    redoc_url="/redoc"     # ReDoc 文档地址
)

# ========================================
# 5. API 路由定义
# ========================================

@app.get("/", tags=["根路由"])
def root():
    """API 根路由 - 返回基本信息"""
    return {
        "message": "欢迎使用用户管理系统 API",
        "version": "1.0.0",
        "docs": "/docs"
    }

@app.get("/users", response_model=List[UserResponse], tags=["用户管理"])
def list_users(
    skip: int = 0,
    limit: int = 10,
    db: dict = Depends(get_db)
):
    """
    获取用户列表
    
    Args:
        skip: 跳过前 N 条记录（用于分页）
        limit: 返回最大记录数（用于分页）
        db: 数据库连接（依赖注入）
    
    Returns:
        List[UserResponse]: 用户列表
    """
    # 从内存数据库获取用户
    users = list(fake_users_db.values())
    
    # 手动转换为 UserResponse（排除敏感字段）
    result = []
    for user in users[skip : skip + limit]:
        result.append(UserResponse(
            id=user.id,
            username=user.username,
            email=user.email,
            is_active=user.is_active,
            created_at=user.created_at
        ))
    
    return result

@app.get("/users/{user_id}", response_model=UserResponse, tags=["用户管理"])
def get_user(user_id: int):
    """
    获取单个用户信息
    
    Args:
        user_id: 用户 ID（路径参数，自动验证为整数）
    
    Returns:
        UserResponse: 用户信息
    
    Raises:
        HTTPException: 用户不存在时返回 404
    """
    if user_id not in fake_users_db:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=f"用户 ID {user_id} 不存在"
        )
    
    user = fake_users_db[user_id]
    return UserResponse(
        id=user.id,
        username=user.username,
        email=user.email,
        is_active=user.is_active,
        created_at=user.created_at
    )

@app.post("/users", response_model=UserResponse, status_code=status.HTTP_201_CREATED, tags=["用户管理"])
def create_user(user: UserCreate):
    """
    创建新用户
    
    Args:
        user: 创建用户的请求数据（自动验证）
    
    Returns:
        UserResponse: 创建成功的用户信息
    
    Raises:
        HTTPException: 邮箱已存在时返回 400
    """
    global user_id_counter
    
    # 检查邮箱是否已存在
    for existing_user in fake_users_db.values():
        if existing_user.email == user.email:
            raise HTTPException(
                status_code=status.HTTP_400_BAD_REQUEST,
                detail=f"邮箱 {user.email} 已被注册"
            )
    
    # 创建新用户（实际项目应 hash 密码）
    new_user = UserInDB(
        id=user_id_counter,
        username=user.username,
        email=user.email,
        hashed_password=f"hashed_{user.password}",  # 应使用 proper hashing
        is_active=user.is_active,
        created_at=datetime.now()
    )
    
    fake_users_db[user_id_counter] = new_user
    user_id_counter += 1
    
    # 返回非敏感信息
    return UserResponse(
        id=new_user.id,
        username=new_user.username,
        email=new_user.email,
        is_active=new_user.is_active,
        created_at=new_user.created_at
    )

@app.put("/users/{user_id}", response_model=UserResponse, tags=["用户管理"])
def update_user(user_id: int, user_update: UserUpdate):
    """
    更新用户信息（完整更新）
    
    Args:
        user_id: 用户 ID
        user_update: 更新的字段（可选）
    
    Returns:
        UserResponse: 更新后的用户信息
    """
    if user_id not in fake_users_db:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=f"用户 ID {user_id} 不存在"
        )
    
    user = fake_users_db[user_id]
    
    # 只更新提供的字段
    if user_update.username is not None:
        user.username = user_update.username
    if user_update.email is not None:
        user.email = user_update.email
    if user_update.is_active is not None:
        user.is_active = user_update.is_active
    
    return UserResponse(
        id=user.id,
        username=user.username,
        email=user.email,
        is_active=user.is_active,
        created_at=user.created_at
    )

@app.delete("/users/{user_id}", status_code=status.HTTP_204_NO_CONTENT, tags=["用户管理"])
def delete_user(user_id: int):
    """
    删除用户
    
    Args:
        user_id: 用户 ID
    
    Returns:
        None: 成功删除返回 204 No Content
    """
    if user_id not in fake_users_db:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=f"用户 ID {user_id} 不存在"
        )
    
    del fake_users_db[user_id]
    return None

@app.get("/users/me", response_model=UserResponse, tags=["用户管理"])
def read_current_user(current_user: UserResponse = Depends(get_current_user)):
    """
    获取当前登录用户信息
    
    Args:
        current_user: 从依赖注入获取的当前用户
    
    Returns:
        UserResponse: 当前用户信息
    """
    return UserResponse(
        id=current_user.id,
        username=current_user.username,
        email=current_user.email,
        is_active=current_user.is_active,
        created_at=current_user.created_at
    )

# ========================================
# 6. 运行应用
# ========================================

if __name__ == "__main__":
    uvicorn.run(
        "main:app",      # 模块:应用实例
        host="0.0.0.0",  # 监听地址
        port=8000,       # 端口
        reload=True      # 开发模式自动重载
    )
```

### 6.2 文件上传与下载

```python
from fastapi import FastAPI, File, UploadFile, HTTPException
from fastapi.responses import FileResponse
import shutil
import os
from pathlib import Path

app = FastAPI()

UPLOAD_DIR = Path("uploads")
UPLOAD_DIR.mkdir(exist_ok=True)

@app.post("/upload")
async def upload_file(file: UploadFile = File(..., description="要上传的文件")):
    """
    上传文件
    
    Args:
        file: 上传的文件对象
    
    Returns:
        dict: 上传结果信息
    """
    # 验证文件类型
    allowed_types = {"image/jpeg", "image/png", "application/pdf"}
    if file.content_type not in allowed_types:
        raise HTTPException(400, f"不支持的文件类型: {file.content_type}")
    
    # 保存文件
    file_path = UPLOAD_DIR / file.filename
    with file_path.open("wb") as buffer:
        shutil.copyfileobj(file.file, buffer)
    
    return {
```

## 9 WebSocket 支持

WebSocket 是一种在单个 TCP 连接上进行全双工通信的协议，适用于实时聊天、实时推送、在线游戏等场景。FastAPI 原生支持 WebSocket，基于 Starlette 的 WebSocket 实现。

### 9.1 基础 WebSocket

```python
from fastapi import FastAPI, WebSocket, WebSocketDisconnect
from typing import List
import json

app = FastAPI()

# WebSocket 连接管理器
class ConnectionManager:
    """
    管理活跃的 WebSocket 连接
    
    维护一个连接列表，支持广播消息给所有客户端
    """
    
    def __init__(self):
        # 存储所有活跃的连接
        self.active_connections: List[WebSocket] = []
    
    async def connect(self, websocket: WebSocket):
        """
        接受新的 WebSocket 连接
        
        Args:
            websocket: WebSocket 连接对象
        """
        await websocket.accept()
        self.active_connections.append(websocket)
    
    def disconnect(self, websocket: WebSocket):
        """
        移除断开的连接
        
        Args:
            websocket: WebSocket 连接对象
        """
        self.active_connections.remove(websocket)
    
    async def send_personal_message(self, message: dict, websocket: WebSocket):
        """
        发送消息给单个客户端
        
        Args:
            message: 要发送的消息（字典）
            websocket: 目标连接
        """
        await websocket.send_text(json.dumps(message))
    
    async def broadcast(self, message: dict):
        """
        广播消息给所有客户端
        
        Args:
            message: 要广播的消息（字典）
        """
        for connection in self.active_connections:
            await connection.send_text(json.dumps(message))

# 创建全局连接管理器实例
manager = ConnectionManager()

@app.websocket("/ws/{client_id}")
async def websocket_endpoint(websocket: WebSocket, client_id: str):
    """
    WebSocket 端点 - 处理实时通信
    
    Args:
        websocket: WebSocket 连接对象
        client_id: 客户端标识符
    
    典型场景：
    - 实时聊天应用
    - 股票行情推送
    - 在线协作工具
    """
    # 1. 接受连接
    await manager.connect(websocket)
    
    try:
        # 2. 循环接收消息
        while True:
            # 接收客户端发送的消息（阻塞等待）
            data = await websocket.receive_text()
            message = json.loads(data)
            
            # 3. 处理消息
            response = {
                "client_id": client_id,
                "message": message.get("text", ""),
                "timestamp": "2024-01-01T00:00:00Z"
            }
            
            # 4. 广播给所有人（包括发送者）
            await manager.broadcast(response)
            
    except WebSocketDisconnect:
        # 5. 处理连接断开
        manager.disconnect(websocket)
        await manager.broadcast({
            "type": "disconnect",
            "client_id": client_id,
            "message": "用户已离开"
        })
```

### 9.2 WebSocket 认证

WebSocket 连接建立时无法使用标准 HTTP 认证，需要在连接时传递认证信息：

```python
from fastapi import FastAPI, WebSocket, Depends, HTTPException
from fastapi.security import HTTPBearer
from typing import Optional

app = FastAPI()
security = HTTPBearer()

async def get_current_user(websocket: WebSocket) -> dict:
    """
    从 WebSocket 连接中提取并验证用户信息
    
    Args:
        websocket: WebSocket 连接对象
    
    Returns:
        dict: 用户信息
    
    Raises:
        HTTPException: 认证失败
    """
    # 从 query 参数或 header 获取 token
    # 推荐：使用 query 参数传递 token
    token = websocket.query_params.get("token")
    
    if not token:
        # 尝试从 header 获取
        token = websocket.headers.get("Authorization", "").replace("Bearer ", "")
    
    if not token or token != "valid_token":
        await websocket.close(code=4001, reason="未授权")
        raise HTTPException(status_code=401, detail="无效的认证凭据")
    
    return {"user_id": 1, "username": "test_user"}
```

### 9.3 实际使用场景

**场景：实时聊天服务器**

```python
from fastapi import FastAPI, WebSocket
from pydantic import BaseModel
from typing import Dict, List
from datetime import datetime
import json

app = FastAPI()

# 简单的聊天室实现
class ChatRoom:
    """聊天室 - 管理消息历史和连接"""
    
    def __init__(self, room_id: str):
        self.room_id = room_id
        self.connections: Dict[str, WebSocket] = {}
        self.message_history: List[dict] = []
    
    async def join(self, user_id: str, websocket: WebSocket):
        """用户加入房间"""
        await websocket.accept()
        self.connections[user_id] = websocket
        
        # 发送历史消息
        await websocket.send_text(json.dumps({
            "type": "history",
            "messages": self.message_history[-50:]  # 最近50条
        }))
        
        # 广播用户加入
        await self.broadcast({
            "type": "system",
            "message": f"用户 {user_id} 加入了房间"
        })
    
    async def leave(self, user_id: str):
        """用户离开房间"""
        if user_id in self.connections:
            del self.connections[user_id]
            await self.broadcast({
                "type": "system",
                "message": f"用户 {user_id} 离开了房间"
            })
    
    async def receive_message(self, user_id: str, content: str):
        """接收并广播消息"""
        message = {
            "user_id": user_id,
            "content": content,
            "timestamp": datetime.now().isoformat()
        }
        self.message_history.append(message)
        
        # 只保留最近1000条消息
        if len(self.message_history) > 1000:
            self.message_history = self.message_history[-1000:]
        
        await self.broadcast({
            "type": "message",
            **message
        })
    
    async def broadcast(self, message: dict):
        """广播消息给房间内所有人"""
        for conn in self.connections.values():
            await conn.send_text(json.dumps(message))

# 房间管理
chat_rooms: Dict[str, ChatRoom] = {}

@app.websocket("/chat/{room_id}/{user_id}")
async def chat_websocket(websocket: WebSocket, room_id: str, user_id: str):
    """
    聊天 WebSocket 端点
    
    Args:
        room_id: 房间 ID
        user_id: 用户 ID
    """
    # 获取或创建房间
    if room_id not in chat_rooms:
        chat_rooms[room_id] = ChatRoom(room_id)
    
    room = chat_rooms[room_id]
    await room.join(user_id, websocket)
    
    try:
        while True:
            data = await websocket.receive_text()
            message = json.loads(data)
            
            if message.get("type") == "message":
                await room.receive_message(user_id, message.get("content", ""))
    except Exception:
        await room.leave(user_id)
```

### **Go 对比**

Go 中通常使用 gorilla/websocket 库实现 WebSocket：

```go
// Go 实现（gorilla/websocket）
var upgrader = websocket.Upgrader{
    CheckOrigin: func(r *http.Request) bool { return true },
}

func handler(w http.ResponseWriter, r *http.Request) {
    conn, err := upgrader.Upgrade(w, r, nil)
    if err != nil {
        return
    }
    
    for {
        // 读取消息
        _, message, err := conn.ReadMessage()
        if err != nil {
            break
        }
        
        // 写入消息
        err = conn.WriteMessage(websocket.TextMessage, message)
        if err != nil {
            break
        }
    }
}
```

**主要差异**：

| 特性 | Go (gorilla/websocket) | FastAPI |
|------|------------------------|---------|
| 连接管理 | 手动管理 | 依赖注入 + 管理器模式 |
| 并发安全 | 需使用 channel/mutex | 异步自动处理 |
| 认证 | 手动从 request 获取 | 依赖注入系统 |
| 类型安全 | 手动处理 | Pydantic 自动验证 |

```python
    使用 Celery 任务队列处理，不阻塞主请求
    
    Args:
        request: 邮件请求数据
    
    Returns:
        任务 ID，用于查询任务状态
    """
    # 触发 Celery 任务
    task = send_email_celery_task.delay(
        email=request.email,
        subject=request.subject,
        body=request.body
    )
    
    return {
        "task_id": task.id,
        "status": "task submitted",
        "message": "邮件发送任务已提交"
    }

@app.post("/generate-report")
async def generate_report(request: ReportRequest):
    """
    异步生成报表
    
    Args:
        request: 报表请求数据
    
    Returns:
        任务 ID
    """
    task = generate_report_celery_task.delay(
        user_id=request.user_id,
        report_type=request.report_type
    )
    
    return {
        "task_id": task.id,
        "status": "processing",
        "message": "报表正在生成中，请使用 task_id 查询结果"
    }

@app.get("/task/{task_id}")
async def get_task_status(task_id: str):
    """
    查询任务状态
    
    Args:
        task_id: Celery 任务 ID
    
    Returns:
        任务状态和结果
    """
    # AsyncResult 用于获取任务状态和结果
    from celery.result import AsyncResult
    
    result = AsyncResult(task_id, app=celery_app)
    
    response = {
        "task_id": task_id,
        "status": result.state,  # PENDING, STARTED, SUCCESS, FAILURE
    }
    
    # 如果任务完成，添加结果
    if result.ready():
        if result.successful():
            response["result"] = result.result
        else:
            response["error"] = str(result.info)
    
    return response

# ========================================
# 4. 启动 Celery Worker
# ========================================

# 命令行启动 worker:
# celery -A main.celery_app worker --loglevel=info --pool=solo

# 注意：
# - 需要先启动 Redis: redis-server
# - 然后启动 Celery worker
# - 最后启动 FastAPI 应用
```

### 10.3 BackgroundTask vs Celery 选择

| 特性 | BackgroundTask | Celery |
|------|----------------|--------|
| 复杂度 | 简单，无需额外依赖 | 需要安装配置 Redis/RabbitMQ |
| 可靠性 | 仅适合轻量任务 | 支持重试、任务状态追踪 |
| 扩展性 | 单进程内 | 支持多机器分布式 |
| 任务监控 | 无 | Flower 监控面板 |
| 适用场景 | 日志、简单通知 | 邮件发送、报表生成、图像处理 |

### **Go 对比**

Go 中通常使用 Channel 或第三方任务队列：

```go

        processJob(job)
    }
}
```

```go

    // 异步执行
    asyncResult, _ := task.ApplyAsync([]interface{}{email, subject})
}
```

**主要差异**：
- BackgroundTask 类似于 Go 的 goroutine 处理
- Celery 类似于 Go 的 machinery 或 asynq 库
- FastAPI 的依赖注入与 Celery 任务自然集成

## 12 GraphQL 支持

GraphQL 是 Facebook 开发的 API 查询语言，相比 REST API，它允许客户端精确指定需要的数据字段。FastAPI 支持多种 GraphQL 库，最常用的是 Strawberry 和 Ariadne。

### 12.1 Strawberry（推荐）

Strawberry 是现代 Python GraphQL 库，支持类型提示和 dataclass：

```python
# graphql_strawberry.py
from fastapi import FastAPI
from strawberry import Schema, Field
from strawberry.types import Info
from typing import List, Optional
from datetime import datetime
import strawberry

app = FastAPI()

# ========================================
# 1. 定义 GraphQL 模型
# ========================================

@strawberry.type(description="用户类型")
class User:
    """用户 GraphQL 类型"""
    
    id: int
    username: str
    email: str
    is_active: bool
    
    # 计算字段（Resolver）
    @strawberry.field(description="用户全名")
    def full_name(self) -> str:
        return f"{self.username} (ID: {self.id})"

@strawberry.input(description="创建用户输入")
class CreateUserInput:
    """创建用户的输入类型"""
    username: str
    email: str
    password: str

@strawberry.input(description="更新用户输入")
class UpdateUserInput:
    """更新用户的输入类型"""
    username: Optional[str] = None
    email: Optional[str] = None
    is_active: Optional[bool] = None

@strawberry.type(description="商品类型")
class Item:
    """商品 GraphQL 类型"""
    id: int
    name: str
    price: float
    quantity: int

# ========================================
# 2. 模拟数据库
# ========================================

class MockDatabase:
    """模拟数据库"""
    
    def __init__(self):
        self.users = {
            1: User(id=1, username="alice", email="alice@example.com", is_active=True),
            2: User(id=2, username="bob", email="bob@example.com", is_active=False),
        }
        self.items = {
            1: Item(id=1, name="Laptop", price=999.99, quantity=10),
            2: Item(id=2, name="Mouse", price=29.99, quantity=100),
        }
        self.user_id_counter = 3
        self.item_id_counter = 3
    
    def get_user(self, user_id: int) -> Optional[User]:
        return self.users.get(user_id)
    
    def list_users(self) -> List[User]:
        return list(self.users.values())
    
    def create_user(self, input: CreateUserInput) -> User:
        user = User(
            id=self.user_id_counter,
            username=input.username,
            email=input.email,
            is_active=True
        )
        self.users[self.user_id_counter] = user
        self.user_id_counter += 1
        return user
    
    def update_user(self, user_id: int, input: UpdateUserInput) -> Optional[User]:
        user = self.users.get(user_id)
        if not user:
            return None
        
        if input.username:
            user.username = input.username
        if input.email:
            user.email = input.email
        if input.is_active is not None:
            user.is_active = input.is_active
        
        return user
    
    def delete_user(self, user_id: int) -> bool:
        if user_id in self.users:
            del self.users[user_id]
            return True
        return False
    
    def get_item(self, item_id: int) -> Optional[Item]:
        return self.items.get(item_id)
    
    def list_items(self) -> List[Item]:
        return list(self.items.values())

db = MockDatabase()

# ========================================
# 3. 定义 Resolver（查询和变更）
# ========================================

@strawberry.type
class Query:
    """GraphQL 查询类型"""
    
    @strawberry.field(description="根据 ID 获取用户")
    def user(self, id: int) -> Optional[User]:
        """获取单个用户"""
        return db.get_user(id)
    
    @strawberry.field(description="获取所有用户")
    def users(self) -> List[User]:
        """获取用户列表"""
        return db.list_users()
    
    @strawberry.field(description="根据 ID 获取商品")
    def item(self, id: int) -> Optional[Item]:
        """获取单个商品"""
        return db.get_item(id)
    
    @strawberry.field(description="获取所有商品")
    def items(self) -> List[Item]:
        """获取商品列表"""
        return db.list_items()
    
    @strawberry.field(description="搜索用户")
    def search_users(self, query: str) -> List[User]:
        """搜索用户（根据用户名）"""
        return [
            user for user in db.list_users()
            if query.lower() in user.username.lower()
        ]

@strawberry.type
class Mutation:
    """GraphQL 变更类型"""
    
    @strawberry.mutation(description="创建用户")
    def create_user(self, input: CreateUserInput) -> User:
        """创建新用户"""
        return db.create_user(input)
    
    @strawberry.mutation(description="更新用户")
    def update_user(self, id: int, input: UpdateUserInput) -> Optional[User]:
        """更新用户信息"""
        return db.update_user(id, input)
    
    @strawberry.mutation(description="删除用户")
    def delete_user(self, id: int) -> bool:
        """删除用户"""
        return db.delete_user(id)

# ========================================
# 4. 创建 Schema 并集成到 FastAPI
# ========================================

# 创建 Schema
schema = Schema(
    query=Query,
    mutation=Mutation,
    # 配置 GraphQL playground
    playground=True,
    # 启用调试模式
    debug=True
)

# 创建 GraphQL 路由
```

### 12.2 Ariadne（Schema-First）

Ariadne 采用 Schema-First 方式，先定义 SDL 再实现 resolver：

```python
# graphql_ariadne.py
from fastapi import FastAPI
from ariadne import (
    QueryType, 
    MutationType, 
    make_executable_schema,
    ObjectType
)
from ariadne.fastapi import GraphQL
from typing import Optional, List

app = FastAPI()

# ========================================
# 1. 定义 GraphQL SDL（Schema Definition Language）
# ========================================

type_defs = """
    type User {
        id: Int!
        username: String!
        email: String!
        isActive: Boolean!
    }
    
    type Item {
        id: Int!
        name: String!
        price: Float!
    }
    
    input CreateUserInput {
        username: String!
        email: String!
    }
    
    type Query {
        user(id: Int!): User
        users: [User!]!
        items: [Item!]!
    }
    
    type Mutation {
        createUser(input: CreateUserInput!): User!
        deleteUser(id: Int!): Boolean!
    }
"""

# ========================================
# 2. 定义 Resolver
# ========================================

# 查询 Resolver
@QueryType.field("user")
def resolve_user(obj, info, id: int):
    """获取单个用户"""
    return {"id": id, "username": f"user_{id}", "email": f"user{id}@example.com", "isActive": True}

@QueryType.field("users")
def resolve_users(obj, info):
    """获取所有用户"""
    return [
        {"id": 1, "username": "alice", "email": "alice@example.com", "isActive": True},
        {"id": 2, "username": "bob", "email": "bob@example.com", "isActive": False},
    ]

@QueryType.field("items")
def resolve_items(obj, info):
    """获取所有商品"""
    return [
        {"id": 1, "name": "Laptop", "price": 999.99},
        {"id": 2, "name": "Mouse", "price": 29.99},
    ]

# 变更 Resolver
@MutationType.field("createUser")
def resolve_create_user(obj, info, input: dict):
    """创建用户"""
    return {"id": 3, "username": input["username"], "email": input["email"], "isActive": True}

@MutationType.field("deleteUser")
def resolve_delete_user(obj, info, id: int):
    """删除用户"""
    return True

# ========================================
# 3. 创建 Schema 并集成
# ========================================

# 创建可执行的 Schema
schema = make_executable_schema(type_defs, [QueryType, MutationType])

# 创建 GraphQL 路由
graphql_app = GraphQL(schema, debug=True)

# 注册到 FastAPI
app.mount("/graphql", graphql_app)
```

### 12.3 GraphQL vs REST 对比

| 特性 | REST API | GraphQL |
|------|----------|---------|
| 数据获取 | 固定端点返回全部数据 | 客户端指定需要字段 |
| 端点数量 | 每个资源一个端点 | 单个端点处理所有查询 |
| 过度获取 | 常见（获取多余字段） | 避免（精确获取） |
| 缓存 | HTTP 缓存 | 需手动实现 |
| 学习曲线 | 低 | 中等 |
| 类型安全 | 需额外工具 | 内置 SDL 强类型 |

### **Go 对比**

Go 中可以使用 gqlgen 或 graphql-go-tools：

```go
// Go 使用 gqlgen
type Resolver struct{}

func (r *Resolver) User(ctx context.Context, id int) (*User, error) {
    return &User{ID: id, Name: "test"}, nil
}

// Schema (schema.graphqls)
type Query {
    user(id: Int!): User
}
```

**主要差异**：
- Strawberry 类似 Go 的 gqlgen（代码优先）
- Ariadne 类似 Go 的 graphql-go-tools（Schema 优先）
- 两者都支持完整的 GraphQL 特性（Query、Mutation、Subscription）

```python
"""

    email: EmailStr
    is_active: bool = True

class UserCreate(BaseModel):
    """创建用户请求"""
    username: str
    email: EmailStr
    password: str

    """登录请求"""

    password: str

        username=user_data.username,
        email=user_data.email,

    user_id_counter += 1
```

```python
# services/order_service/main.py
"""
订单服务
负责：订单创建、查询、管理
端口：8002
"""

from fastapi import FastAPI, HTTPException
from pydantic import BaseModel
from typing import List, Optional
from datetime import datetime
import uvicorn
import httpx

app = FastAPI(title="订单服务", version="1.0.0")

# ========================================
# 数据模型
# ========================================

class OrderItem(BaseModel):
    """订单项"""
    item_id: int
    quantity: int
    price: float

class OrderCreate(BaseModel):
    """创建订单请求"""
    user_id: int
    items: List[OrderItem]

class Order(BaseModel):
    """订单模型"""
    id: int
    user_id: int
    items: List[OrderItem]
    total_amount: float
    status: str
    created_at: datetime

# ========================================
# 模拟数据库
# ========================================

orders_db = {}
order_id_counter = 1

# ========================================
# 服务间通信
# ========================================

async def verify_user_exists(user_id: int) -> bool:
    """
    验证用户是否存在（调用用户服务）
    
    Args:
        user_id: 用户 ID
    
    Returns:
        bool: 用户是否存在
    """
    try:
        async with httpx.AsyncClient() as client:
            response = await client.get(f"http://localhost:8001/users/{user_id}")
            return response.status_code == 200
    except httpx.RequestError:
        # 服务不可用时返回 False
        return False

async def get_item_price(item_id: int) -> Optional[float]:
    """
    获取商品价格（调用商品服务）
    
    Args:
        item_id: 商品 ID
    
    Returns:
        float: 商品价格
    """
    try:
        async with httpx.AsyncClient() as client:
            response = await client.get(f"http://localhost:8003/items/{item_id}")
            if response.status_code == 200:
                return response.json().get("price")
    except httpx.RequestError:
        pass
    return None

# ========================================
# API 端点
# ========================================

@app.get("/health")
def health_check():
    """健康检查"""
    return {"status": "healthy", "service": "order-service"}

@app.post("/orders", response_model=Order)
async def create_order(order_data: OrderCreate):
    """
    创建订单
    
    涉及服务间调用：
    1. 验证用户存在
    2. 获取商品价格
    """
    global order_id_counter
    
    # 1. 验证用户
    user_exists = await verify_user_exists(order_data.user_id)
    if not user_exists:
        raise HTTPException(status_code=400, detail="用户不存在")
    
    # 2. 计算订单金额并验证商品
    total_amount = 0.0
    validated_items = []
    
    for item in order_data.items:
        # 从商品服务获取价格
        price = await get_item_price(item.item_id)
        if price is None:
            raise HTTPException(status_code=400, detail=f"商品 {item.item_id} 不存在")
        
        # 使用商品服务的价格（防止前端篡改）
        item.price = price
        total_amount += price * item.quantity
        validated_items.append(item)
    
    # 3. 创建订单
    order = Order(
        id=order_id_counter,
        user_id=order_data.user_id,
        items=validated_items,
        total_amount=total_amount,
        status="pending",
        created_at=datetime.now()
    )
    
    orders_db[order_id_counter] = order
    order_id_counter += 1
    
    return order

@app.get("/orders/{order_id}", response_model=Order)
def get_order(order_id: int):
    """获取订单详情"""
    if order_id not in orders_db:
        raise HTTPException(status_code=404, detail="订单不存在")
    return orders_db[order_id]

@app.get("/orders/user/{user_id}", response_model=List[Order])
def get_user_orders(user_id: int):
    """获取用户的所有订单"""
    return [order for order in orders_db.values() if order.user_id == user_id]

if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8002)
```

### 13.3 API 网关

```python
# api_gateway/main.py
"""
API 网关
负责：统一入口、路由转发、认证、限流
端口：8000
"""

from fastapi import FastAPI, HTTPException, Request, Header
from fastapi.responses import JSONResponse
from fastapi.middleware.cors import CORSMiddleware
import httpx
from typing import Optional
import uvicorn
from datetime import datetime

app = FastAPI(title="API Gateway", version="1.0.0")

# CORS 中间件
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# ========================================
# 服务地址配置
# ========================================

SERVICES = {
    "user": "http://localhost:8001",
    "order": "http://localhost:8002",
    "item": "http://localhost:8003",
}

# ========================================
# 认证中间件
# ========================================

async def verify_token(authorization: Optional[str] = Header(None)) -> dict:
    """
    验证 JWT Token
    
    Args:
        authorization: Authorization header
    
    Returns:
        dict: 用户信息
    
    Raises:
        HTTPException: 认证失败
    """
    if not authorization:
        raise HTTPException(status_code=401, detail="缺少认证信息")
    
    # 解析 Bearer token
    if not authorization.startswith("Bearer "):
        raise HTTPException(status_code=401, detail="无效的认证格式")
    
    token = authorization.replace("Bearer ", "")
    
    # 简化验证：实际应使用 JWT 库验证
    if not token.startswith("token_"):
        raise HTTPException(status_code=401, detail="无效的 Token")
    
    user_id = token.replace("token_", "")
    return {"user_id": int(user_id)}

# ========================================
# 路由代理
# ========================================

@app.api_route("/{service}/{path:path}", methods=["GET", "POST", "PUT", "DELETE"])
async def proxy(
    service: str,
    path: str,
    request: Request,
    authorization: Optional[str] = Header(None)
):
    """
    通用代理路由
    
    Args:
        service: 服务名称（user/order/item）
        path: 目标路径
        request: 原始请求
        authorization: 认证信息
    
    Returns:
        代理服务的响应
    """
    # 1. 验证服务存在
    if service not in SERVICES:
        raise HTTPException(status_code=404, detail=f"服务 {service} 不存在")
    
    # 2. 获取目标 URL
    target_url = f"{SERVICES[service]}/{path}"
    
    # 3. 转发请求
    async with httpx.AsyncClient() as client:
        try:
            # 读取请求体
            body = await request.body()
            
            # 构建转发请求
            headers = dict(request.headers)
            if authorization:
                headers["Authorization"] = authorization
            
            response = await client.request(
                method=request.method,
                url=target_url,
                headers=headers,
                content=body,
                params=request.query_params,
            )
            
            # 4. 返回响应
            return JSONResponse(
                content=response.json() if response.text else {},
                status_code=response.status_code,
            )
            
        except httpx.RequestError as e:
            raise HTTPException(
                status_code=503,
                detail=f"服务 {service} 不可用: {str(e)}"
            )

# ========================================
# 聚合端点（减少客户端请求）
# ========================================

@app.get("/aggregated/user/{user_id}")
async def get_user_with_orders(
    user_id: int,
    authorization: Optional[str] = Header(None)
):
    """
    获取用户信息及其订单（聚合端点）
    
    客户端只需一次请求即可获取完整数据
    """
    async with httpx.AsyncClient() as client:
        # 并发请求用户和订单信息
        user_response, orders_response = await asyncio.gather(
            client.get(f"{SERVICES['user']}/users/{user_id}", headers={"Authorization": authorization or ""}),
            client.get(f"{SERVICES['order']}/orders/user/{user_id}", headers={"Authorization": authorization or ""}),
        )
        
        return {
            "user": user_response.json() if user_response.status_code == 200 else None,
            "orders": orders_response.json() if orders_response.status_code == 200 else [],
        }

# ========================================
# 健康检查
# ========================================

@app.get("/health")
async def health_check():
    """检查网关和下游服务健康状态"""
    import asyncio
    
    health_status = {
        "gateway": "healthy",
        "services": {}
    }
    
    async with httpx.AsyncClient() as client:
        # 并发检查所有服务
        tasks = [
            client.get(f"{url}/health")
            for url in SERVICES.values()
        ]
        
        results = await asyncio.gather(*tasks, return_exceptions=True)
        
        for (name, url), result in zip(SERVICES.items(), results):
            if isinstance(result, Exception):
                health_status["services"][name] = "unhealthy"
            elif result.status_code == 200:
                health_status["services"][name] = "healthy"
            else:
                health_status["services"][name] = "degraded"
    
    return health_status

import asyncio

if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

```python
# 服务注册示例（使用 Consul）
import consul
import requests

def register_service():
    """注册服务到 Consul"""
    c = consul.Consul(host="localhost", port=8500)
    
    # 注册服务
    c.agent.service.register(
        "user-service",
        service_id="user-service-1",
        address="localhost",
        port=8001,
        check=consul.Check.http(
            "http://localhost:8001/health",
            interval="10s"
        )
    )

def discover_service(service_name: str) -> str:
    """从 Consul 发现服务地址"""
    c = consul.Consul(host="localhost", port=8500)
    
    # 查询健康服务
    _, services = c.health.service(service_name, passing=True)
    
    if services:
        # 返回第一个健康服务
        return f"http://{services[0]['Service']['Address']}:{services[0]['Service']['Port']}"
    
    raise Exception(f"No healthy instance of {service_name}")
```

### 1.5 常用标准库速查

## 1 技术简介

**Python标准库（Standard Library）**是Python安装时自带的模块集合，体现了Python「Batteries Included（自带电池）」的哲学。与Go语言的最小标准库不同，Python提供了丰富而全面的标准库，涵盖文件操作、网络通信、数据处理、系统交互等几乎所有编程领域。

**核心特点**：
- **丰富性**：200+个标准模块，开箱即用
- **稳定性**：由核心团队维护，API稳定可靠
- **一致性**：遵循统一的设计规范和编码风格
- **跨平台**：Windows、Linux、macOS通用

**与Go的对比**：
- Go：最小标准库，依赖第三方包（go get）
- Python：最大标准库，减少外部依赖

## 2 技术背景

### 为什么会出现

Python诞生于1991年，比Go（2009年）早了18年。在漫长的发展过程中，Python不断吸纳各种实用功能进入标准库：

| 时期 | 主要添加内容 |
|------|-------------|
| Python 1.x | 基础系统接口、字符串处理 |
| Python 2.x | 网络编程、数据库、多线程 |
| Python 3.x | 异步IO、类型注解、f-string |

### Python 2 vs 3差异

Go开发者需要注意的关键变化：

| 特性 | Python 2 | Python 3 | Go等价 |
|------|----------|----------|--------|
| print | `print "hello"` | `print("hello")` | `fmt.Println()` |
| 字符串 | 默认字节 | 默认Unicode | `string` |
| 除法 | `5/2 = 2` | `5/2 = 2.5` | 整数/浮点区分 |
| range | `xrange()` | `range()` | 切片操作 |

### 与Go的设计理念差异

```
Go: 最小化标准库，依赖显式管理
     ↓
    优点：依赖清晰，二进制小
    缺点：需要大量第三方包

Python: 最大化标准库， batteries included
     ↓
    优点：开箱即用，开发快
    缺点：安装包较大
```

#### os模块 - 操作系统接口

**Go等价**: `os` 包

```python
# 环境变量
```

#### pathlib模块 - 面向对象路径

**Go等价**: `path/filepath` 包（但更现代化）

```python

# 创建路径对象

# 路径操作
print(p.name)       # file.txt（文件名）
print(p.parent)     # /home/user/documents（父目录）
print(p.suffix)     # .txt（扩展名）
```

#### json模块 - JSON处理

**Go等价**: `encoding/json` 包

```python
import json

# Python对象 → JSON字符串
# Go: json.Marshal()
data = {"name": "张三", "age": 25}
json_str = json.dumps(data, ensure_ascii=False)
print(json_str)  # {"name": "张三", "age": 25}

# JSON字符串 → Python对象

parsed = json.loads(json_str)

# 读写文件

with open("data.json", "w", encoding="utf-8") as f:
```

**Go等价**: `regexp` 包

#### collections模块 - 高级容器

**Go等价**: 需要自行实现或使用第三方库


## 6 Python代码实现与逐行解析

### 6.1 文件操作

```python
"""
文件操作示例 - 对比Go实现
"""

import os
import shutil
from pathlib import Path

# ============ 文件读写 ============
# Go: os.ReadFile() / os.WriteFile()

# Python: 读取文件（推荐with语句）
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()  # 读取全部
    lines = f.readlines()  # 读取为列表

# Python: 写入文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello, World!\n")
    f.writelines(["Line 1\n", "Line 2\n"])

# ============ 目录操作 ============
# Go: os.MkdirAll() / os.RemoveAll()

# 创建目录
os.makedirs("path/to/dir", exist_ok=True)  # 递归创建

# 删除目录
shutil.rmtree("path/to/dir")  # 递归删除
os.rmdir("empty_dir")  # 删除空目录

# ============ 文件信息 ============
# Go: os.Stat()

from pathlib import Path

p = Path("file.txt")
if p.exists():
    print(f"Size: {p.stat().st_size} bytes")
    print(f"Modified: {p.stat().st_mtime}")

# ============ 遍历目录 ============
# Go: filepath.WalkDir()

# 方法1: os.walk
for root, dirs, files in os.walk("/path"):
    for file in files:
        filepath = os.path.join(root, file)
        print(filepath)

# 方法2: pathlib（更现代）
for filepath in Path("/path").rglob("*.py"):  # 递归查找.py文件
    print(filepath)
```

**代码解析：**
- `with open("data.txt", "r", encoding="utf-8") as f`: with语句是上下文管理器，自动关闭文件。open()返回文件对象，"r"是读取模式（类似Go的os.O_RDONLY），encoding指定编码避免乱码。Go等价: `data, err := os.ReadFile("file.txt")`。
- `os.makedirs("path/to/dir", exist_ok=True)`: 递归创建目录。Go等价: `os.MkdirAll()`。
- `shutil.rmtree("path/to/dir")`: 递归删除目录。Go等价: `os.RemoveAll()`。
- `Path("/path").rglob("*.py")`: 递归glob匹配。Go等价: `filepath.WalkDir()`。

### 6.3 系统命令执行

```python
"""
执行系统命令 - 对比Go的os/exec
"""

import subprocess
import shlex

# ============ 简单执行 ============
# Go: exec.Command().Output()

result = subprocess.run(
    ["ls", "-la"],
    capture_output=True,
    text=True
)
print(result.stdout)
print(result.returncode)

# ============ 安全执行（避免shell注入）============
# Go: exec.Command() with separate args

cmd = "ls"
args = ["-la", "/home"]
result = subprocess.run([cmd] + args, capture_output=True, text=True)

# ============ 管道操作 ============
# Go: cmd1 | cmd2

p1 = subprocess.Popen(["ls", "-la"], stdout=subprocess.PIPE)
p2 = subprocess.Popen(["grep", "py"], stdin=p1.stdout, stdout=subprocess.PIPE)
p1.stdout.close()
output = p2.communicate()[0]

# ============ 超时控制 ============
# Go: context.WithTimeout()

try:
    result = subprocess.run(
        ["sleep", "10"],
        timeout=5,  # 5秒超时
        capture_output=True
    )
except subprocess.TimeoutExpired:
    print("命令超时")
```

**代码解析：**
- `subprocess.run(["ls", "-la"], capture_output=True, text=True)`: 执行命令并捕获输出。Go等价: `exec.Command().Output()`。
- `timeout=5`: 超时控制。Go等价: `context.WithTimeout()`。
- 使用列表传参避免shell注入，比shell=True更安全。

```python
import csv
```

### 8.1 脚本自动化

```python
#!/usr/bin/env python3
"""批量重命名工具 - 类似Go的CLI工具"""

import os
import sys
from pathlib import Path

def batch_rename(pattern, replacement, directory="."):
    """批量重命名文件"""
    for filepath in Path(directory).glob(pattern):
        new_name = filepath.name.replace(pattern, replacement)
        new_path = filepath.parent / new_name
        filepath.rename(new_path)
        print(f"Renamed: {filepath} -> {new_path}")

if __name__ == "__main__":
    if len(sys.argv) < 3:
        print("Usage: python rename.py <pattern> <replacement>")
        sys.exit(1)
    
    batch_rename(sys.argv[1], sys.argv[2])
```

### 8.3 数据处理管道

```python
"""数据处理管道 - ETL示例"""

import json
import csv
from datetime import datetime
from pathlib import Path

def extract(file_path):
    """从JSON提取数据"""
    with open(file_path, 'r', encoding='utf-8') as f:
        return json.load(f)

def transform(data):
    """转换数据"""
    for item in data:
        item['processed_at'] = datetime.now().isoformat()
        item['upper_name'] = item['name'].upper()
    return data

def load(data, output_path):
    """加载到CSV"""
    with open(output_path, 'w', newline='', encoding='utf-8') as f:
        writer = csv.DictWriter(f, fieldnames=data[0].keys())
        writer.writeheader()
        writer.writerows(data)

# 运行管道
data = extract("input.json")
data = transform(data)
load(data, "output.csv")
```

```python
import hashlib

def hash_password(password, salt=None):
    """简单密码哈希（仅示例，生产环境用bcrypt）"""
    if salt is None:
        salt = secrets.token_hex(16)
    pwd_hash = hashlib.pbkdf2_hmac(
        'sha256',
        password.encode(),
        salt.encode(),
        100000  # 迭代次数
    )
    return f"{salt}${pwd_hash.hex()}"

def verify_password(password, stored):
    """验证密码"""
    salt, pwd_hash = stored.split('$')
    new_hash = hashlib.pbkdf2_hmac(
        'sha256',
        password.encode(),
        salt.encode(),
        100000
    )
    return hmac.compare_digest(pwd_hash, new_hash.hex())

# 使用示例

print(f"Hashed: {hashed}")
print(f"Valid: {verify_password('mypassword', hashed)}")
```

- `secrets.token_urlsafe(16)`: 生成密码学安全的随机字符串。Go等价: `rand.Reader` + 编码。

- `textwrap.fill(text, width=20)`: 将文本按宽度填充。Go需要手动实现。

```python
# 场景1：API认证中的Basic Auth
def generate_basic_auth(username, password):
    """生成Basic Auth头部值"""
    import base64

    encoded = base64.b64encode(credentials.encode()).decode()

def encode_url_token(data):
    """URL安全编码token"""
    import base64
    import urllib.parse
    # 先Base64编码，再URL编码
    b64 = base64.urlsafe_b64encode(data.encode()).decode()
    return urllib.parse.quote(b64)

def decode_url_token(encoded):
    """解码token"""
    import base64
    import urllib.parse
    b64 = urllib.parse.unquote(encoded)
    return base64.urlsafe_b64decode(b64).decode()

token = encode_url_token('{"user_id": 123}')

# ============ 大文件流式编码 ============
# Go: NewEncoder() / NewDecoder()

def base64_encode_stream(input_file, output_file):
    """流式Base64编码（内存友好）"""
    with open(input_file, 'rb') as fin:
        with open(output_file, 'w') as fout:
            # 分块读取和编码
            while True:
                chunk = fin.read(3 * 1024)  # 3KB块（Base64要求3的倍数）
                if not chunk:
                    break
                encoded = base64.b64encode(chunk).decode()
                fout.write(encoded)

def base64_decode_stream(input_file, output_file):
    """流式Base64解码"""
    with open(input_file, 'r') as fin:
        with open(output_file, 'wb') as fout:
            # 一次读取多行（每行是Base64编码的块）
            while True:
                chunk = fin.read(4 * 1024)  # 4KB块
                if not chunk:
                    break
                decoded = base64.b64decode(chunk)
                fout.write(decoded)
```

**代码解析：**
- `base64.b64encode(data)`: Base64编码。Go等价: `base64.StdEncoding.EncodeToString()`。
- `base64.urlsafe_b64encode()`: URL安全的Base64（`+`→`-`, `/`→`_`）。Go等价: `base64.URLEncoding`。
- `binascii.hexlify(data)`: 十六进制编码。Go等价: `hex.Encode()`。
- `binascii.crc32(data)`: CRC32校验码。Go等价: `hash/crc32.New()`。
- `base64.standard_b64encode()`: 标准Base64（带`=`填充）。
- 流式编码适合大文件，每次处理小块避免内存溢出。

# 12.6 Multi-Agent协作系统

## 1 技术简介

Multi-Agent协作系统是一个展示多个智能Agent如何分工合作完成复杂任务的完整框架。该系统模拟了真实软件开发团队的工作模式，包含项目经理Agent、开发Agent、测试Agent和文档Agent四种角色。系统通过消息队列和通信总线实现Agent间的信息传递，支持任务分配、进度跟踪、结果汇总等协作功能。这种多Agent架构能够将复杂任务拆解并行处理，大幅提升任务执行效率。

## 2 技术背景

随着AI应用场景的复杂化，单一Agent已难以满足复杂任务的需求。多Agent系统通过让多个专业化的Agent协同工作，每个Agent专注于特定类型的任务，实现了"分而治之"的策略。这种架构灵感来源于人类团队协作模式，不同角色的Agent各司其职，通过标准化通信协议协调工作，类似于现代软件开发中的敏捷团队模式。

## 3 核心概念

**Agent（智能体）**：系统中独立运行的实体，每个Agent有特定的角色和能力。系统定义了BaseAgent基类，所有具体Agent都继承自该基类。

**Message（消息）**：Agent间通信的基本单位，包含发送者、接收者、消息类型和内容。消息类型包括任务分配、任务结果、查询、响应、广播和错误等。

**Task（任务）**：待执行的工作单元，包含任务ID、类型、描述、优先级和状态等信息。任务状态在待处理、已分配、进行中、已完成、失败之间流转。

**CommunicationBus（通信总线）**：Agent间消息传递的中枢，实现发布-订阅模式，支持广播和点对点通信。

**TaskScheduler（任务调度器）**：管理任务的创建、分配和执行，支持优先级调度策略。

## 4 工作原理

系统启动时，首先创建通信总线和任务调度器，然后初始化四种Agent并注册到总线上。项目经理Agent维护任务队列并管理其他Agent的注册。当提交项目时，项目经理根据任务类型将任务分配给对应的Agent——开发任务分配给开发Agent，测试任务分配给测试Agent，文档任务分配给文档Agent。

每个Agent都有一个消息队列，用于接收来自总线的消息。Agent处理消息时，根据消息类型执行相应的操作：开发Agent收到任务后生成代码，测试Agent执行测试用例，文档Agent生成文档。处理完成后，Agent通过通信总线将结果发送回项目经理。项目经理收集所有结果，汇总成最终的项目报告。整个过程中，任务调度器负责协调任务的调度执行，通信总线确保消息可靠传递。

## 5 技术实现

系统采用面向对象设计，主要类包括：Message和Task数据类定义消息和任务的结构；BaseAgent是所有Agent的抽象基类，定义通用接口；ProjectManagerAgent、DeveloperAgent、TesterAgent、DocumenterAgent分别实现项目经理、开发、测试、文档四种角色；CommunicationBus实现消息传递机制；TaskScheduler管理任务调度；MultiAgentSystem整合所有组件提供统一入口。系统使用Queue存储消息，使用List管理任务历史，使用threading实现简单的并发处理。

# ============ 消息和任务数据结构 ============

class MessageType(Enum):
    """消息类型枚举：定义Agent间通信的消息类型"""
    TASK_ASSIGN = "task_assign"       # 任务分配
    TASK_RESULT = "task_result"       # 任务结果
    QUERY = "query"                   # 查询消息
    RESPONSE = "response"             # 响应消息
    BROADCAST = "broadcast"           # 广播消息
    ERROR = "error"                   # 错误消息


class TaskStatus(Enum):
    """任务状态枚举：定义任务的生命周期状态"""
    PENDING = "pending"               # 待处理
    ASSIGNED = "assigned"             # 已分配
    IN_PROGRESS = "in_progress"       # 进行中
    COMPLETED = "completed"           # 已完成
    FAILED = "failed"                 # 失败


@dataclass
class Message:
    """消息数据类：Agent间通信的基本单位"""
    sender: str                       # 发送者ID
    receiver: str                     # 接收者ID（"broadcast"表示广播）
    msg_type: MessageType             # 消息类型
    content: Dict[str, Any]           # 消息内容
    timestamp: datetime = field(default_factory=datetime.now)  # 时间戳
    msg_id: str = field(default_factory=lambda: f"msg_{int(time.time()*1000)}")  # 消息ID


@dataclass
class Task:
    """任务数据类：描述一个待执行的工作单元"""
    task_id: str                      # 任务唯一标识
    task_type: str                    # 任务类型（development/testing/documentation等）
    description: str                  # 任务描述
    priority: int = 1                 # 优先级（1-5，5最高）
    status: TaskStatus = TaskStatus.PENDING  # 任务状态
    assigned_to: Optional[str] = None # 分配给的Agent
    result: Optional[Any] = None      # 任务结果
    created_at: datetime = field(default_factory=datetime.now)  # 创建时间
    completed_at: Optional[datetime] = None  # 完成时间


# ============ Agent基类 ============

class BaseAgent(ABC):
    """Agent基类：定义所有Agent的通用接口和行为
    
    所有具体的Agent（项目经理、开发、测试、文档）都继承此类
    实现各自的handle_message方法来处理特定类型的任务
    """
    
    def __init__(self, agent_id: str, name: str, agent_type: str):
        """初始化Agent
        
        参数:
            agent_id: Agent唯一标识
            name: Agent名称
            agent_type: Agent类型（manager/developer/tester/documenter）
        """
        self.agent_id = agent_id      # Agent的唯一标识符
        self.name = name              # Agent的显示名称
        self.agent_type = agent_type  # Agent类型
        self.message_queue: Queue = Queue()  # 消息队列，存储接收到的消息
        self.task_history: List[Task] = []   # 任务历史记录
        self.is_active = True         # Agent是否活跃
        self.communication_bus: Optional['CommunicationBus'] = None  # 通信总线引用
    
    def set_communication_bus(self, bus: 'CommunicationBus'):
        """设置通信总线
        
        参数:
            bus: 通信总线实例
        """
        self.communication_bus = bus
    
    def receive_message(self, message: Message):
        """接收消息，存入消息队列
        
        参数:
            message: 接收到的消息对象
        """
        self.message_queue.put(message)
    
    @abstractmethod
    def handle_message(self, message: Message) -> Optional[Message]:
        """处理消息的抽象方法，由子类实现
        
        参数:
            message: 待处理的消息
        返回:
            可选的响应消息
        """
        pass
    
    def process_messages(self):
        """处理消息队列中的所有消息"""
        while not self.message_queue.empty():
            message = self.message_queue.get()
            response = self.handle_message(message)
            if response and self.communication_bus:
                self.communication_bus.send_message(response)
    
    def send_message(self, receiver: str, msg_type: MessageType, content: Dict) -> bool:
        """发送消息给其他Agent
        
        参数:
            receiver: 接收者ID
            msg_type: 消息类型
            content: 消息内容
        返回:
            发送是否成功
        """
        if not self.communication_bus:
            return False
        
        message = Message(
            sender=self.agent_id,
            receiver=receiver,
            msg_type=msg_type,
            content=content
        )
        self.communication_bus.send_message(message)
        return True
    
    def get_status(self) -> Dict[str, Any]:
        """获取Agent状态信息"""
        return {
            "agent_id": self.agent_id,
            "name": self.name,
            "type": self.agent_type,
            "is_active": self.is_active,
            "pending_messages": self.message_queue.qsize(),
            "completed_tasks": len([t for t in self.task_history if t.status == TaskStatus.COMPLETED])
        }


# ============ 项目经理Agent ============

class ProjectManagerAgent(BaseAgent):
    """项目经理Agent：负责任务分配和项目协调
    
    核心职责：
    1. 接收外部任务请求
    2. 分析任务类型并分配给合适的Agent
    3. 跟踪任务进度
    4. 汇总结果
    """
    
    def __init__(self, agent_id: str = "pm_001"):
        """初始化项目经理Agent"""
        super().__init__(agent_id, "项目经理", "manager")
        self.task_queue: List[Task] = []
        self.agents: Dict[str, BaseAgent] = {}
        self.project_status: Dict[str, Any] = {
            "total_tasks": 0,
            "completed_tasks": 0,
            "failed_tasks": 0
        }
    
    def register_agent(self, agent: BaseAgent):
        """注册Agent到项目团队"""
        self.agents[agent.agent_id] = agent
        print(f"[{self.name}] 注册Agent: {agent.name} ({agent.agent_type})")
    
    def create_task(self, description: str, task_type: str, priority: int = 1) -> Task:
        """创建新任务"""
        task_id = f"task_{int(time.time()*1000)}_{random.randint(1000, 9999)}"
        task = Task(
            task_id=task_id,
            task_type=task_type,
            description=description,
            priority=priority
        )
        self.task_queue.append(task)
        self.project_status["total_tasks"] += 1
        print(f"[{self.name}] 创建任务: {task_id} - {description[:30]}...")
        return task
    
    def assign_task(self, task: Task) -> bool:
        """分配任务给合适的Agent"""
        # 根据任务类型选择合适的Agent
        target_type = self._get_target_agent_type(task.task_type)
        
        # 查找该类型的Agent
        for agent_id, agent in self.agents.items():
            if agent.agent_type == target_type and agent.is_active:
                self.send_message(
                    agent_id,
                    MessageType.TASK_ASSIGN,
                    {
                        "task_id": task.task_id,
                        "task_type": task.task_type,
                        "description": task.description,
                        "priority": task.priority
                    }
                )
                task.assigned_to = agent_id
                task.status = TaskStatus.ASSIGNED
                print(f"[{self.name}] 分配任务 {task.task_id} 给 {agent.name}")
                return True
        print(f"[{self.name}] 没有可用的Agent处理任务类型: {task.task_type}")
        return False
    
    def _get_target_agent_type(self, task_type: str) -> str:
        """根据任务类型确定目标Agent类型"""
        mapping = {
            "development": "developer",
            "testing": "tester",
            "documentation": "documenter",
            "code_review": "developer"
        }
        return mapping.get(task_type, "developer")
    
    def handle_message(self, message: Message) -> Optional[Message]:
        """处理接收到的消息"""
        if message.msg_type == MessageType.TASK_RESULT:
            task_id = message.content.get("task_id")
            success = message.content.get("success", False)
            result = message.content.get("result")
            
            # 更新项目状态
            if success:
                self.project_status["completed_tasks"] += 1
                print(f"[{self.name}] 收到任务完成: {task_id}")
            else:
                self.project_status["failed_tasks"] += 1
                print(f"[{self.name}] 收到任务失败: {task_id}, 原因: {result}")
            
            # 如果所有任务完成，生成报告
            total = self.project_status["total_tasks"]
            completed = self.project_status["completed_tasks"]
            if completed + self.project_status["failed_tasks"] >= total and total > 0:
                print(f"\n[{self.name}] 所有任务已完成!")
                print(f"  总任务: {total}")
                print(f"  成功: {completed}")
                print(f"  失败: {self.project_status['failed_tasks']}")
        
        elif message.msg_type == MessageType.QUERY:
            query_type = message.content.get("query_type")
            if query_type == "project_status":
                return Message(
                    sender=self.agent_id,
                    receiver=message.sender,
                    msg_type=MessageType.RESPONSE,
                    content={"project_status": self.project_status}
                )
        
        return None
    
    def run_project(self):
        """运行项目：分配所有待处理任务"""
        print(f"\n[{self.name}] 开始执行项目...")
        for task in self.task_queue:
            if task.status == TaskStatus.PENDING:
                self.assign_task(task)

            {"name": "性能测试", "passed": random.random() > 0.2},
            {"name": "边界测试", "passed": random.random() > 0.15},
        ]
        
        all_passed = all(tc["passed"] for tc in test_cases)
        
        return {
            "description": description,
            "test_cases": test_cases,
            "passed": all_passed,
            "total": len(test_cases),
            "passed_count": sum(1 for tc in test_cases if tc["passed"]),

        }

        """生成测试报告"""

- 总测试数: {len(self.test_results)}
- 通过: {sum(1 for r in self.test_results if r.get('passed'))}
- 失败: {sum(1 for r in self.test_results if not r.get('passed'))}
"""
        return report

# ============ Agent通信机制 ============

class CommunicationBus:
    """通信总线：Agent间消息传递的中枢
    
    所有Agent通过通信总线发送和接收消息
    实现发布-订阅模式，支持广播和点对点通信
    """
    
    def __init__(self):
        """初始化通信总线"""
        self.agents: Dict[str, BaseAgent] = {}
        self.message_history: List[Message] = []
        self.max_history: int = 1000
    
    def register_agent(self, agent: BaseAgent):
        """注册Agent到通信总线"""
        self.agents[agent.agent_id] = agent
        agent.set_communication_bus(self)
        print(f"[通信总线] 注册Agent: {agent.name} ({agent.agent_id})")
    
    def send_message(self, message: Message):
        """发送消息"""
        # 记录消息历史
        self.message_history.append(message)
        if len(self.message_history) > self.max_history:
            self.message_history.pop(0)
        
        # 判断是广播还是点对点
        if message.receiver == "broadcast":
            for agent_id, agent in self.agents.items():
                if agent_id != message.sender:
                    agent.receive_message(message)
            print(f"[通信总线] 广播消息: {message.msg_type.value}")
        else:
            if message.receiver in self.agents:
                self.agents[message.receiver].receive_message(message)
                print(f"[通信总线] 消息从 {message.sender} -> {message.receiver}")
            else:
                print(f"[通信总线] 警告: 未找到接收者 {message.receiver}")
    
    def get_message_stats(self) -> Dict:
        """获取消息统计信息"""
        return {
            "total_messages": len(self.message_history),
            "registered_agents": len(self.agents),
            "message_types": {
                msg_type.value: sum(1 for m in self.message_history if m.msg_type == msg_type)
                for msg_type in MessageType
            }
        }

class TaskScheduler:
    """任务调度系统：管理任务的创建、分配和执行
    
    核心功能：
    1. 任务优先级排序
    2. 任务依赖管理
    3. 任务状态跟踪
    4. 调度策略执行
    """
    
    def __init__(self, communication_bus: CommunicationBus):
        """初始化任务调度器"""
        self.bus = communication_bus
        self.tasks: Dict[str, Task] = {}
        self.task_queue: List[Task] = []
        self.scheduling_strategy: str = "priority"
    
    def add_task(self, task: Task):
        """添加任务到调度队列"""
        self.tasks[task.task_id] = task
        self.task_queue.append(task)
        print(f"[调度器] 添加任务: {task.task_id} (优先级: {task.priority})")
    
    def get_next_task(self) -> Optional[Task]:
        """获取下一个待执行的任务"""
        if not self.task_queue:
            return None
        
        # 根据调度策略选择任务
        if self.scheduling_strategy == "priority":
            self.task_queue.sort(key=lambda t: t.priority, reverse=True)
        elif self.scheduling_strategy == "fifo":
            pass
        
        for task in self.task_queue:
            if task.status == TaskStatus.PENDING:
                return task
        
        return None
    
    def schedule_task(self, task: Task, target_agent_id: str) -> bool:
        """调度任务到指定Agent"""
        message = Message(
            sender="scheduler",
            receiver=target_agent_id,
            msg_type=MessageType.TASK_ASSIGN,
            content={
                "task_id": task.task_id,
                "task_type": task.task_type,
                "description": task.description,
                "priority": task.priority
            }
        )
        
        task.status = TaskStatus.ASSIGNED
        task.assigned_to = target_agent_id
        
        self.bus.send_message(message)
        print(f"[调度器] 调度任务 {task.task_id} -> {target_agent_id}")
        
        return True
    
    def update_task_status(self, task_id: str, status: TaskStatus, result: Any = None):
        """更新任务状态"""
        if task_id in self.tasks:
            task = self.tasks[task_id]
            task.status = status
            if result:
                task.result = result
            if status == TaskStatus.COMPLETED:
                task.completed_at = datetime.now()
            print(f"[调度器] 更新任务 {task_id} 状态: {status.value}")
    
    def get_task_statistics(self) -> Dict:
        """获取任务统计信息"""
        tasks = list(self.tasks.values())
        return {
            "total": len(tasks),
            "pending": sum(1 for t in tasks if t.status == TaskStatus.PENDING),
            "assigned": sum(1 for t in tasks if t.status == TaskStatus.ASSIGNED),
            "in_progress": sum(1 for t in tasks if t.status == TaskStatus.IN_PROGRESS),
            "completed": sum(1 for t in tasks if t.status == TaskStatus.COMPLETED),
            "failed": sum(1 for t in tasks if t.status == TaskStatus.FAILED)
        }
    
    def run_scheduling_cycle(self):
        """执行一次调度周期"""
        print("\n[调度器] 开始调度周期...")
        
        task = self.get_next_task()
        while task:
            target_type = self._get_agent_type_for_task(task.task_type)
            
            for agent_id, agent in self.bus.agents.items():
                if agent.agent_type == target_type and agent.is_active:
                    self.schedule_task(task, agent_id)
                    break
            
            task = self.get_next_task()
    
    def _get_agent_type_for_task(self, task_type: str) -> str:
        """根据任务类型获取Agent类型"""
        mapping = {
            "development": "developer",
            "testing": "tester",
            "documentation": "documenter"
        }
        return mapping.get(task_type, "developer")

## 7 常见问题

**问题一：消息丢失**
在并发场景下，消息可能在传输过程中丢失。建议添加消息确认机制，接收方收到消息后发送确认回执。

**问题二：Agent响应超时**
某些任务执行时间较长，可能导致整个系统阻塞。可以为每个Agent设置独立线程处理消息，实现真正的并行。

**问题三：任务依赖处理不足**
当前系统不支持任务间的依赖关系，后续任务可能在前置任务完成前就开始执行。需要扩展TaskScheduler以支持依赖图。

**问题四：状态一致性**
当系统出现异常时，Agent状态可能出现不一致。建议引入持久化机制，定期保存状态快照。

# 11.2 Agent任务执行系统

## 1 技术简介

Agent任务执行系统是一种能够自主理解任务目标、规划执行步骤、调用工具并处理执行结果的智能系统。与传统的被动响应式对话不同，Agent具备主动思考和行动的能力，能够根据任务需求选择合适的工具，观察工具返回的结果，并根据结果调整后续行动，形成一个完整的"思考-行动-观察"循环。系统通过提示工程定义Agent的行为规范，使用大语言模型作为推理引擎，实现了对复杂任务的自动化处理。

## 2 技术背景

大语言模型展现出强大的推理和生成能力，但其本身只能输出文本，无法直接执行具体操作。Agent架构的提出解决了这一问题，它将LLM作为"大脑"，赋予其使用"工具"的能力。ReAct（Reasoning + Acting）范式的出现进一步强化了Agent的推理和行动能力，使其能够处理多步骤的复杂任务。当前的Agent系统已广泛应用于自动化工作流、智能助手、代码生成等领域。

## 3 核心概念

**Agent（智能体）**：具备自主推理和行动能力的智能程序，能够根据目标自主规划路径并执行操作，而非仅仅响应用户输入。

**工具（Tools）**：Agent可以调用的外部功能模块，如计算器、搜索API、文件读写接口等。每个工具都有明确的输入输出规范。

**记忆（Memory）**：Agent的上下文保持机制，存储历史交互记录，使Agent能够跟踪多轮对话的状态和任务进度。

**提示工程（Prompt Engineering）**：通过精心设计的系统提示来定义Agent的角色、行为规范和输出格式，引导模型按预期方式工作。

**ReAct循环**：Reasoning（推理）→Acting（行动）→Observing（观察）的循环模式，Agent在此循环中不断推进任务执行。

**任务规划（Task Planning）**：将复杂任务分解为可执行的子任务序列，确定执行顺序和依赖关系。

**人机交互（Human-in-the-loop）**：在关键决策点引入人类确认或输入，确保系统行为符合预期。

## 4 工作原理

Agent任务执行系统的工作流程是一个迭代的"思考-行动-观察"循环。系统启动时，首先加载系统提示，其中定义了Agent的角色、可用的工具列表以及输出格式规范。当用户提出任务后，Agent开始第一轮推理，分析任务需求并决定下一步行动。

在行动阶段，Agent从工具列表中选择合适的工具，准备相应的参数，并执行工具调用。工具执行完成后返回结果，Agent进入观察阶段，分析执行结果并评估是否达到任务目标。如果任务已完成，Agent输出"完成:"格式的结果；如果未完成，则进入下一轮推理，根据观察到的结果调整策略，继续执行下一个行动。这个过程持续进行，直到任务完成或达到最大迭代次数。

对于复杂任务，系统首先进行任务规划，将大任务分解为多个子任务，然后逐个执行。在执行过程中，可以通过记忆系统保存上下文，通过重试机制处理临时错误，通过人机交互在关键节点获取用户确认。整个系统支持同步和异步执行模式，通过状态追踪实时监控任务进度。

## 5 系统架构

系统采用分层架构设计，从底层到顶层依次为：

**工具层（Tool Layer）**：提供各种可调用工具的实现，包括搜索、计算、代码执行、文件操作等。每个工具都是独立的函数，具有统一的接口定义。

**Agent层（Agent Layer）**：核心执行逻辑，包括BaseAgent抽象基类和各种具体Agent实现。负责任务规划、工具调用、结果处理等核心功能。

**记忆层（Memory Layer）**：管理Agent的上下文信息，包括短期记忆（当前任务状态）和长期记忆（历史任务记录）。

**编排层（Orchestration Layer）**：负责任务调度、状态管理、错误处理和重试逻辑。

**接口层（Interface Layer）**：提供REST API和WebSocket接口，支持同步和异步任务提交、状态查询、实时进度推送。

class TaskStatus(Enum):
    """任务状态枚举类
    
    定义任务在整个生命周期中的所有可能状态
    """
    PENDING = "pending"           # 任务已创建，等待执行
    PLANNING = "planning"         # 正在规划任务步骤
    EXECUTING = "executing"       # 正在执行中
    WAITING_HUMAN = "waiting_human"  # 等待人类确认
    COMPLETED = "completed"       # 任务已完成
    FAILED = "failed"             # 任务执行失败
    CANCELLED = "cancelled"       # 任务被取消


@dataclass
class Task:
    """任务数据类
    
    用于存储任务的完整信息，包括ID、描述、状态、执行计划等
    """
    id: str                                           # 任务唯一标识符
    description: str                                  # 任务描述文本
    status: TaskStatus = TaskStatus.PENDING           # 当前任务状态
    plan: List[Dict] = field(default_factory=list)    # 任务执行计划（子任务列表）
    current_step: int = 0                             # 当前执行的步骤索引
    result: Optional[str] = None                      # 任务最终结果
    error: Optional[str] = None                       # 错误信息（如果失败）
    created_at: datetime = field(default_factory=datetime.now)  # 创建时间
    updated_at: datetime = field(default_factory=datetime.now)  # 更新时间
    memory: List[Dict] = field(default_factory=list)  # 任务执行记忆
    human_input: Optional[str] = None                 # 人类输入（人机交互用）
    
    def to_dict(self) -> Dict:
        """将任务转换为字典格式，用于JSON序列化"""
        return {
            "id": self.id,
            "description": self.description,
            "status": self.status.value,
            "plan": self.plan,
            "current_step": self.current_step,
            "result": self.result,
            "error": self.error,
            "created_at": self.created_at.isoformat(),
            "updated_at": self.updated_at.isoformat(),
            "memory": self.memory
        }


class BaseAgent(ABC):
    """Agent抽象基类
    
    定义所有Agent必须实现的接口和方法，确保系统的一致性和可扩展性
    子类需要实现工具注册、提示生成、响应解析等核心方法
    """
    
    def __init__(self, name: str, description: str = ""):
        """初始化Agent
        
        Args:
            name: Agent的名称
            description: Agent的描述信息
        """
        self.name = name
        self.description = description
        self.tools: Dict[str, Callable] = {}     # 工具注册表
        self.max_iterations = 10                 # 最大迭代次数
        self.temperature = 0.7                   # LLM温度参数
        
    @abstractmethod
    def register_tools(self) -> None:
        """注册Agent可用的工具
        
        子类必须实现此方法，定义自己的工具集
        """
        pass
    
    @abstractmethod
    def get_system_prompt(self) -> str:
        """获取系统提示词
        
        子类必须实现此方法，返回Agent的系统提示
        """
        pass
    
    @abstractmethod
    def parse_response(self, response: str) -> Dict:
        """解析LLM响应
        
        从LLM输出中提取思考过程、行动指令等信息
        """
        pass
    
    def add_tool(self, name: str, func: Callable, description: str = "") -> None:
        """注册单个工具
        
        Args:
            name: 工具名称
            func: 工具函数
            description: 工具描述
        """
        self.tools[name] = func
        logger.info(f"注册工具: {name}")
    
    def execute_tool(self, tool_name: str, params: Any) -> str:
        """执行工具
        
        根据工具名称调用对应的工具函数，处理执行异常
        
        Args:
            tool_name: 工具名称
            params: 工具参数
            
        Returns:
            工具执行结果
        """
        if tool_name not in self.tools:
            return f"错误：未知工具 '{tool_name}'"
        
        try:
            # 根据参数类型调用工具
            if params:
                return self.tools[tool_name](params)
            else:
                return self.tools[tool_name]()
        except Exception as e:
            logger.error(f"工具执行错误: {tool_name} - {str(e)}")
            return f"工具执行错误: {str(e)}"


class ToolRegistry:
    """工具注册中心
    
    提供全局工具管理功能，支持工具的注册、查询、批量操作等
    可用于多个Agent之间共享工具
    """
    
    def __init__(self):
        """初始化工具注册中心"""
        self._tools: Dict[str, Callable] = {}
        self._descriptions: Dict[str, str] = {}
        self._metadata: Dict[str, Dict] = {}
    
    def register(self, name: str, func: Callable, description: str = "", 
                 parameters: Dict = None, category: str = "general") -> None:
        """注册工具
        
        Args:
            name: 工具唯一名称
            func: 工具函数
            description: 工具功能描述
            parameters: 参数模式定义（用于生成工具说明）
            category: 工具分类
        """
        self._tools[name] = func
        self._descriptions[name] = description
        self._metadata[name] = {
            "parameters": parameters or {},
            "category": category,
            "registered_at": datetime.now().isoformat()
        }
        logger.info(f"工具注册成功: {name} (分类: {category})")
    
    def get(self, name: str) -> Optional[Callable]:
        """获取工具函数"""
        return self._tools.get(name)
    
    def get_description(self, name: str) -> str:
        """获取工具描述"""
        return self._descriptions.get(name, "")
    
    def list_tools(self, category: str = None) -> List[str]:
        """列出所有工具
        
        Args:
            category: 可选，按分类过滤
            
        Returns:
            工具名称列表
        """
        if category:
            return [name for name, meta in self._metadata.items() 
                   if meta.get("category") == category]
        return list(self._tools.keys())
    
    def generate_schema(self) -> List[Dict]:
        """生成工具模式定义
        
        返回符合OpenAI工具调用格式的工具定义
        """
        schema = []
        for name, meta in self._metadata.items():
            schema.append({
                "type": "function",
                "function": {
                    "name": name,
                    "description": self._descriptions.get(name, ""),
                    "parameters": meta.get("parameters", {
                        "type": "object",
                        "properties": {"input": {"type": "string"}},
                        "required": []
                    })
                }
            })
        return schema


class TaskPlanner:
    """任务规划器
    
    负责将复杂任务分解为可执行的子任务序列
    支持依赖关系分析、并行优化、计划验证等功能
    """
    
    def __init__(self, llm_client: OpenAI):
        """初始化任务规划器
        
        Args:
            llm_client: OpenAI客户端实例
        """
        self.client = llm_client
    
    def create_plan(self, task: str, available_tools: List[str]) -> List[Dict]:
        """创建任务执行计划
        
        分析任务需求，将大任务分解为多个子任务，并确定执行顺序
        
        Args:
            task: 任务描述
            available_tools: 可用工具列表
            
        Returns:
            子任务计划列表，每个子任务包含描述、工具、参数等信息
        """
        tools_str = ", ".join(available_tools)
        
        planning_prompt = f"""请分析以下任务，并将其分解为具体的执行步骤。

任务: {task}

可用工具: {tools_str}

请按以下JSON格式输出执行计划：
{{
    "steps": [
        {{
            "step": 1,
            "description": "步骤描述",
            "tool": "工具名称",
            "params": "工具参数",
            "depends_on": []  # 依赖的前置步骤编号
        }}
    ],
    "reasoning": "整体规划思路"
}}

确保每个步骤：
1. 描述清晰、具体
2. 有明确的工具调用
3. 正确设置依赖关系
"""
        
        try:
            response = self.client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=[{"role": "user", "content": planning_prompt}],
                temperature=0.3,
                max_tokens=2000
            )
            
            content = response.choices[0].message.content
            # 提取JSON部分
            json_match = re.search(r'\{[\s\S]*\}', content)
            if json_match:
                plan_data = json.loads(json_match.group())
                return plan_data.get("steps", [])
            
            # 解析失败时返回默认计划
            return [{"step": 1, "description": task, "tool": None, "params": task, "depends_on": []}]
            
        except Exception as e:
            logger.error(f"任务规划失败: {str(e)}")
            return [{"step": 1, "description": task, "tool": None, "params": task, "depends_on": []}]
    
    def validate_plan(self, plan: List[Dict]) -> bool:
        """验证执行计划的合理性
        
        检查计划中是否存在循环依赖、孤立步骤等问题
        
        Args:
            plan: 执行计划
            
        Returns:
            计划是否有效
        """
        if not plan:
            return False
            
        # 检查步骤编号连续性
        step_nums = {s.get("step") for s in plan}
        expected = set(range(1, len(plan) + 1))
        if step_nums != expected:
            logger.warning("步骤编号不连续")
            return False
        
        # 检查依赖关系是否存在循环
        def has_cycle(step_num: int, visited: set, plan_dict: Dict) -> bool:
            if step_num in visited:
                return True
            visited.add(step_num)
            step = plan_dict.get(step_num, {})
            for dep in step.get("depends_on", []):
                if has_cycle(dep, visited.copy(), plan_dict):
                    return True
            return False
        
        plan_dict = {s["step"]: s for s in plan}
        for step in plan:
            if has_cycle(step["step"], set(), plan_dict):
                logger.warning("计划存在循环依赖")
                return False
        
        return True


class MemoryManager:
    """记忆管理器
    
    负责管理Agent的上下文记忆，包括短期记忆（当前任务）和长期记忆（历史经验）
    支持记忆的存储、检索、总结、清理等功能
    """
    
    def __init__(self, max_short_memory: int = 10, max_long_memory: int = 100):
        """初始化记忆管理器
        
        Args:
            max_short_memory: 短期记忆最大容量
            max_long_memory: 长期记忆最大容量
        """
        self.max_short_memory = max_short_memory
        self.max_long_memory = max_long_memory
        self.short_memory: List[Dict] = []    # 短期记忆：当前任务的执行历史
        self.long_memory: List[Dict] = []     # 长期记忆：历史任务经验
    
    def add(self, memory_type: str, data: Dict) -> None:
        """添加记忆
        
        Args:
            memory_type: 记忆类型 ("short" 或 "long")
            data: 记忆数据
        """
        if memory_type == "short":
            self.short_memory.append({
                **data,
                "timestamp": datetime.now().isoformat()
            })
            # 超过容量时，尝试压缩最旧的记忆
            if len(self.short_memory) > self.max_short_memory:
                self._compress_short_memory()
        elif memory_type == "long":
            self.long_memory.append({
                **data,
                "timestamp": datetime.now().isoformat()
            })
            # 超过容量时，移除最旧的记忆
            if len(self.long_memory) > self.max_long_memory:
                self.long_memory.pop(0)
    
    def get_short_memory(self) -> List[Dict]:
        """获取短期记忆"""
        return self.short_memory.copy()
    
    def get_long_memory(self, query: str = None, limit: int = 5) -> List[Dict]:
        """获取长期记忆
        
        Args:
            query: 可选的查询关键词
            limit: 返回数量限制
            
        Returns:
            匹配的记忆列表
        """
        if not query:
            return self.long_memory[-limit:]
        
        # 简单的关键词匹配
        results = []
        for mem in reversed(self.long_memory):
            if any(query.lower() in str(v).lower() for v in mem.values()):
                results.append(mem)
                if len(results) >= limit:
                    break
        return results
    
    def clear_short_memory(self) -> None:
        """清空短期记忆"""
        self.short_memory.clear()
    
    def _compress_short_memory(self) -> None:
        """压缩短期记忆
        
        将多个记忆条目合并为一个摘要，减少内存占用
        """
        if len(self.short_memory) < 3:
            return
        
        # 保留首尾记忆，压缩中间部分
        compressed = [self.short_memory[0]]
        middle = self.short_memory[1:-1]
        
        # 生成摘要
        summary = {
            "type": "compressed",
            "summary": f"中间执行了 {len(middle)} 个步骤",
            "last_result": middle[-1].get("observation", "") if middle else ""
        }
        compressed.append(summary)
        compressed.append(self.short_memory[-1])
        
        self.short_memory = compressed
    
    def get_context_for_llm(self, max_entries: int = 5) -> str:
        """生成供LLM使用的上下文文本
        
        将记忆转换为LLM可以理解的文本格式
        
        Args:
            max_entries: 最大记忆条目数
            
        Returns:
            格式化的上下文文本
        """
        context_parts = []
        
        # 添加长期记忆（相关经验）
        long_mem = self.get_long_memory(limit=3)
        if long_mem:
            context_parts.append("## 历史经验")
            for mem in long_mem:
                context_parts.append(f"- {mem.get('description', '任务')}: {mem.get('result', '')}")
        
        # 添加短期记忆（最近执行步骤）
        recent_steps = self.short_memory[-max_entries:]
        if recent_steps:
            context_parts.append("\n## 最近执行步骤")
            for step in recent_steps:
                obs = step.get("observation", "")[:100]
                context_parts.append(f"- {step.get('response', '')[:100]} → {obs}")
        
        return "\n".join(context_parts)


class RetryHandler:
    """重试处理器
    
    提供自动重试功能，处理临时性错误如网络超时、服务不可用等
    支持指数退避策略、错误类型判断、最大重试次数控制
    """
    
    def __init__(self, max_retries: int = 3, base_delay: float = 1.0, 
                 max_delay: float = 60.0):
        """初始化重试处理器
        
        Args:
            max_retries: 最大重试次数
            base_delay: 基础延迟时间（秒）
            max_delay: 最大延迟时间（秒）
        """
        self.max_retries = max_retries
        self.base_delay = base_delay
        self.max_delay = max_delay
    
    async def execute_with_retry(self, func: Callable, *args, **kwargs) -> Any:
        """带重试的异步执行
        
        Args:
            func: 要执行的异步函数
            *args, **kwargs: 函数参数
            
        Returns:
            函数执行结果
            
        Raises:
            最后一次尝试的错误
        """
        last_error = None
        
        for attempt in range(self.max_retries + 1):
            try:
                # 判断是否为异步函数
                if asyncio.iscoroutinefunction(func):
                    return await func(*args, **kwargs)
                else:
                    return func(*args, **kwargs)
                    
            except Exception as e:
                last_error = e
                if attempt < self.max_retries:
                    # 计算延迟时间（指数退避）
                    delay = min(self.base_delay * (2 ** attempt), self.max_delay)
                    logger.warning(f"执行失败，{delay}秒后重试 ({attempt + 1}/{self.max_retries}): {str(e)}")
                    await asyncio.sleep(delay)
                else:
                    logger.error(f"重试次数耗尽，执行失败: {str(e)}")
        
        raise last_error
    
    def execute_with_retry_sync(self, func: Callable, *args, **kwargs) -> Any:
        """带重试的同步执行
        
        Args:
            func: 要执行的同步函数
            *args, **kwargs: 函数参数
            
        Returns:
            函数执行结果
        """
        last_error = None
        
        for attempt in range(self.max_retries + 1):
            try:
                return func(*args, **kwargs)
            except Exception as e:
                last_error = e
                if attempt < self.max_retries:
                    delay = min(self.base_delay * (2 ** attempt), self.max_delay)
                    logger.warning(f"执行失败，{delay}秒后重试 ({attempt + 1}/{self.max_retries}): {str(e)}")
                    import time
                    time.sleep(delay)
                else:
                    logger.error(f"重试次数耗尽，执行失败: {str(e)}")
        
        raise last_error


class HumanInTheLoop:
    """人机交互控制器
    
    在关键决策点暂停执行，等待人类确认或输入
    支持确认请求、选项选择、自由输入等交互模式
    """
    
    def __init__(self):
        """初始化人机交互控制器"""
        self.pending_requests: Dict[str, Dict] = {}
        self.callbacks: Dict[str, Callable] = {}
    
    def request_confirmation(self, task_id: str, message: str, 
                            on_confirm: Callable = None, 
                            on_cancel: Callable = None) -> str:
        """请求用户确认
        
        Args:
            task_id: 任务ID
            message: 确认消息
            on_confirm: 确认后的回调函数
            on_cancel: 取消后的回调函数
            
        Returns:
            请求ID
        """
        request_id = f"confirm_{task_id}_{datetime.now().timestamp()}"
        
        self.pending_requests[request_id] = {
            "type": "confirmation",
            "task_id": task_id,
            "message": message,
            "status": "pending",
            "created_at": datetime.now()
        }
        
        if on_confirm:
            self.callbacks[f"{request_id}_confirm"] = on_confirm
        if on_cancel:
            self.callbacks[f"{request_id}_cancel"] = on_cancel
        
        logger.info(f"请求确认: {request_id} - {message}")
        return request_id
    
    def request_choice(self, task_id: str, message: str, 
                      options: List[str], on_select: Callable = None) -> str:
        """请求用户选择
        
        Args:
            task_id: 任务ID
            message: 选择提示
            options: 选项列表
            on_select: 选择后的回调函数
            
        Returns:
            请求ID
        """
        request_id = f"choice_{task_id}_{datetime.now().timestamp()}"
        
        self.pending_requests[request_id] = {
            "type": "choice",
            "task_id": task_id,
            "message": message,
            "options": options,
            "status": "pending",
            "created_at": datetime.now()
        }
        
        if on_select:
            self.callbacks[f"{request_id}_select"] = on_select
        
        return request_id
    
    def request_input(self, task_id: str, message: str, 
                     on_submit: Callable = None) -> str:
        """请求用户输入
        
        Args:
            task_id: 任务ID
            message: 输入提示
            on_submit: 提交后的回调函数
            
        Returns:
            请求ID
        """
        request_id = f"input_{task_id}_{datetime.now().timestamp()}"
        
        self.pending_requests[request_id] = {
            "type": "input",
            "task_id": task_id,
            "message": message,
            "status": "pending",
            "created_at": datetime.now()
        }
        
        if on_submit:
            self.callbacks[f"{request_id}_submit"] = on_submit
        
        return request_id
    
    def process_response(self, request_id: str, response: Any) -> None:
        """处理用户响应
        
        Args:
            request_id: 请求ID
            response: 用户响应
        """
        if request_id not in self.pending_requests:
            logger.warning(f"未找到请求: {request_id}")
            return
        
        request = self.pending_requests[request_id]
        request["status"] = "completed"
        request["response"] = response
        
        # 触发相应回调
        if request["type"] == "confirmation":
            callback_key = f"{request_id}_confirm" if response else f"{request_id}_cancel"
            if callback_key in self.callbacks:
                self.callbacks[callback_key](response)
        elif request["type"] == "choice":
            if f"{request_id}_select" in self.callbacks:
                self.callbacks[f"{request_id}_select"](response)
        elif request["type"] == "input":
            if f"{request_id}_submit" in self.callbacks:
                self.callbacks[f"{request_id}_submit"](response)
        
        logger.info(f"处理用户响应: {request_id} -> {response}")
    
    def get_pending_requests(self, task_id: str = None) -> List[Dict]:
        """获取待处理的请求
        
        Args:
            task_id: 可选，按任务ID过滤
            
        Returns:
            请求列表
        """
        requests = [r for r in self.pending_requests.values() 
                   if r["status"] == "pending"]
        
        if task_id:
            requests = [r for r in requests if r["task_id"] == task_id]
        
        return requests


class TaskExecutionAgent(BaseAgent):
    """任务执行Agent
    
    功能：
    - 理解用户任务需求
    - 自主规划执行步骤
    - 调用工具完成任务
    - 处理执行过程中的异常
    这是一个完整的Agent实现，支持多工具调用和迭代执行
    """
    def __init__(self, api_key: str = None):
        """初始化任务执行Agent
        
        Args:
            api_key: OpenAI API密钥
        """
        super().__init__("TaskExecutionAgent", "通用任务执行智能体")

        self.client = OpenAI(api_key=api_key or os.getenv("OPENAI_API_KEY"))
        self.memory_manager = MemoryManager()
        self.retry_handler = RetryHandler(max_retries=3)
        
        # 注册默认工具
        self.register_tools()
    
    def register_tools(self) -> None:
        """注册Agent可用的工具集合"""
        
        def calculate(expression: str) -> str:
            """计算器工具：安全地执行数学表达式计算"""
            try:
                allowed_chars = set('0123456789+-*/.() ')
                if all(c in allowed_chars for c in expression):
                    result = eval(expression)
                    return str(result)
                return "错误：表达式包含非法字符"
            except Exception as e:
                return f"计算错误: {str(e)}"
        
        def search(query: str) -> str:
            """搜索工具（模拟）：返回模拟的搜索结果"""
            return f"搜索结果：找到关于'{query}'的相关信息..."
        
        def get_time(_=None) -> str:
            """获取当前时间：返回当前日期时间"""
            return datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        
        def file_read(filepath: str) -> str:
            """读取文件工具：从指定路径读取文件内容"""
            try:
                with open(filepath, 'r', encoding='utf-8') as f:
                    return f.read()[:1000]
            except Exception as e:
                return f"读取失败: {str(e)}"
        
        def file_write(params: str) -> str:
            """写入文件工具：格式为"文件路径|内容" """
            try:
                parts = params.split('|', 1)
                if len(parts) != 2:
                    return "格式错误，使用: 文件路径|内容"
                filepath, content = parts
                with open(filepath, 'w', encoding='utf-8') as f:
                    f.write(content)
                return f"成功写入文件: {filepath}"
            except Exception as e:
                return f"写入失败: {str(e)}"
        
        def execute_code(code: str) -> str:
            """代码执行工具：安全地执行Python代码"""
            try:
                # 创建安全的执行环境
                safe_globals = {"__builtins__": {}}
                result = eval(code, safe_globals, {})
                return str(result) if result is not None else "代码执行完成（无返回值）"
            except SyntaxError:
                try:
                    exec(code, {"__builtins__": {}})
                    return "代码执行完成"
                except Exception as e:
                    return f"执行错误: {str(e)}"
            except Exception as e:
                return f"执行错误: {str(e)}"
        
        def fetch_url(url: str) -> str:
            """网页获取工具：获取指定URL的内容"""
            try:
                import urllib.request
                response = urllib.request.urlopen(url, timeout=10)
                return response.read().decode('utf-8')[:2000]
            except Exception as e:
                return f"获取失败: {str(e)}"
        
        # 注册所有工具
        self.add_tool("calculate", calculate, "计算数学表达式")
        self.add_tool("search", search, "搜索信息")
        self.add_tool("get_time", get_time, "获取当前时间")
        self.add_tool("file_read", file_read, "读取文件内容")

        self.add_tool("execute_code", execute_code, "执行Python代码")
        self.add_tool("fetch_url", fetch_url, "获取网页内容")
    
    def get_system_prompt(self) -> str:
        """获取系统提示词"""
        tool_descriptions = "\n".join([
            f"- {name}: {desc}" 
            for name, desc in [("calculate", "计算数学表达式"), 
                              ("search", "搜索信息"),
                              ("get_time", "获取当前时间"),
                              ("file_read", "读取文件"),
                              ("file_write", "写入文件"),
                              ("execute_code", "执行Python代码"),
                              ("fetch_url", "获取网页内容")]
        ])
        
        return f"""你是一个智能任务执行助手。你可以使用以下工具来完成任务：

{tool_descriptions}

请按以下格式回复：
思考: [你的分析和计划]
行动: [工具名] [参数]
或
完成: [任务结果]
或
确认: [需要用户确认的内容]

注意：
1. 每次只执行一个工具
2. 观察工具返回后继续下一步
3. 如果任务完成，使用"完成:"格式回复
4. 需要用户确认时使用"确认:"格式
5. 如果工具参数包含特殊字符，请用引号包裹
"""
    
    def parse_response(self, response: str) -> Dict:
        """解析LLM响应"""
        result = {
            "type": "unknown",
            "thought": "",
            "action": None,
            "action_params": "",
            "completion": None,
            "confirmation": None
        }
        
        # 解析思考
        thought_match = re.search(r'思考:\s*(.+?)(?=\n行动:|\n完成:|\n确认:|$)', response, re.DOTALL)
        if thought_match:
            result["thought"] = thought_match.group(1).strip()
        
        # 解析行动
        action_match = re.search(r'行动:\s*(\w+)\s*(.+)?', response, re.DOTALL)
        if action_match:
            result["type"] = "action"
            result["action"] = action_match.group(1).strip()
            result["action_params"] = action_match.group(2).strip() if action_match.group(2) else ""
        
        # 解析完成
        completion_match = re.search(r'完成:\s*(.+)$', response, re.DOTALL)
        if completion_match:
            result["type"] = "completion"
            result["completion"] = completion_match.group(1).strip()
        
        # 解析确认请求
        confirm_match = re.search(r'确认:\s*(.+)$', response, re.DOTALL)
        if confirm_match:
            result["type"] = "confirmation"
            result["confirmation"] = confirm_match.group(1).strip()
        
        return result
    
    def execute_task(self, task: str) -> str:
        """执行任务（同步版本）
        
        Args:
            task: 任务描述
            
        Returns:
            任务执行结果
        """
        print(f"🎯 任务: {task}\n")
        
        # 清空记忆，开始新任务
        self.memory_manager.clear_short_memory()
        
        # 迭代执行循环
        for iteration in range(self.max_iterations):
            print(f"--- 迭代 {iteration + 1} ---")
            
            # 构建消息
            messages = [
                {"role": "system", "content": self.get_system_prompt()},
                {"role": "user", "content": f"任务: {task}"}
            ]
            
            # 添加记忆上下文
            context = self.memory_manager.get_context_for_llm()
            if context:
                messages.append({"role": "system", "content": f"\n{context}"})
            
            # 添加历史交互
            for mem in self.memory_manager.get_short_memory():
                messages.append({"role": "assistant", "content": mem.get("response", "")})
                messages.append({"role": "user", "content": f"观察: {mem.get('observation', '')}"})
            
            # 调用LLM
            try:
                response = self.client.chat.completions.create(
                    model="gpt-3.5-turbo",
                    messages=messages,
                    temperature=self.temperature
                )
                content = response.choices[0].message.content
            except Exception as e:
                logger.error(f"LLM调用失败: {str(e)}")
                return f"执行失败: {str(e)}"
            
            print(f"🤔 助手:\n{content}\n")
            
            # 解析响应
            parsed = self.parse_response(content)
            
            # 处理完成
            if parsed["type"] == "completion":
                result = parsed["completion"]
                print(f"✅ 任务完成: {result}")
                
                # 保存到长期记忆
                self.memory_manager.add("long", {
                    "description": task,
                    "result": result
                })
                return result
            
            # 处理确认请求
            if parsed["type"] == "confirmation":
                # 简单处理：直接继续执行
                print(f"⚠️ 需要确认: {parsed['confirmation']}")
            
            # 处理行动
            if parsed["type"] == "action" and parsed["action"]:
                tool_name = parsed["action"]
                tool_params = parsed["action_params"]
                
                print(f"🔧 执行: {tool_name}({tool_params})")
                
                # 执行工具（带重试）
                try:
                    observation = self.retry_handler.execute_with_retry_sync(
                        self.execute_tool, tool_name, tool_params
                    )
                except Exception as e:
                    observation = f"工具执行错误: {str(e)}"
                
                print(f"👁️ 观察: {observation}\n")
                
                # 保存到短期记忆
                self.memory_manager.add("short", {
                    "response": content,
                    "observation": observation,
                    "step": iteration + 1
                })
            else:
                print("❌ 无法解析响应，任务中断")
                break
        
        return "任务未完成，达到最大迭代次数"
    
    async def execute_task_async(self, task: str) -> str:
        """执行任务（异步版本）
        
        Args:
            task: 任务描述
            
        Returns:
            任务执行结果
        """
        logger.info(f"异步任务开始: {task}")
        
        self.memory_manager.clear_short_memory()
        
        for iteration in range(self.max_iterations):
            messages = [
                {"role": "system", "content": self.get_system_prompt()},
                {"role": "user", "content": f"任务: {task}"}
            ]
            
            # 添加记忆
            context = self.memory_manager.get_context_for_llm()
            if context:
                messages.append({"role": "system", "content": f"\n{context}"})
            
            for mem in self.memory_manager.get_short_memory():
                messages.append({"role": "assistant", "content": mem.get("response", "")})
                messages.append({"role": "user", "content": f"观察: {mem.get('observation', '')}"})
            
            # 异步调用LLM
            try:
                response = await self.retry_handler.execute_with_retry(
                    self._call_llm_async, messages
                )
                content = response.choices[0].message.content
            except Exception as e:
                logger.error(f"LLM调用失败: {str(e)}")
                return f"执行失败: {str(e)}"
            
            parsed = self.parse_response(content)
            
            if parsed["type"] == "completion":
                result = parsed["completion"]
                self.memory_manager.add("long", {
                    "description": task,
                    "result": result
                })
                return result
            
            if parsed["type"] == "action" and parsed["action"]:
                tool_name = parsed["action"]
                tool_params = parsed["action_params"]
                
                # 异步执行工具
                observation = await self.retry_handler.execute_with_retry(
                    asyncio.to_thread, self.execute_tool, tool_name, tool_params
                )
                
                self.memory_manager.add("short", {
                    "response": content,
                    "observation": observation,
                    "step": iteration + 1
                })
        
        return "任务未完成，达到最大迭代次数"
    
    async def _call_llm_async(self, messages: List[Dict]) -> Any:
        """异步调用LLM"""
        loop = asyncio.get_event_loop()
        return await loop.run_in_executor(
            None,
            lambda: self.client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=messages,
                temperature=self.temperature
            )
        )


class TaskExecutor:
    """任务执行器
    
    负责管理多个任务的执行，支持任务队列、并行执行、状态追踪等功能
    """
    
    def __init__(self, agent: TaskExecutionAgent):
        """初始化任务执行器
        
        Args:
            agent: 任务执行Agent实例
        """
        self.agent = agent
        self.tasks: Dict[str, Task] = {}
        self.task_history: List[Task] = []
        self.status_listeners: List[Callable] = []
    
    def create_task(self, description: str, task_id: str = None) -> Task:
        """创建新任务
        
        Args:
            description: 任务描述
            task_id: 可选的任务ID
            
        Returns:
            创建的任务对象
        """
        task_id = task_id or f"task_{datetime.now().timestamp()}"
        
        task = Task(
            id=task_id,
            description=description,
            status=TaskStatus.PENDING
        )
        
        self.tasks[task_id] = task
        self._notify_status_change(task)
        
        return task
    
    def execute_task_sync(self, task_id: str) -> str:
        """同步执行任务
        
        Args:
            task_id: 任务ID
            
        Returns:
            执行结果
        """
        if task_id not in self.tasks:
            return f"任务不存在: {task_id}"
        
        task = self.tasks[task_id]
        task.status = TaskStatus.EXECUTING
        self._notify_status_change(task)
        
        try:
            result = self.agent.execute_task(task.description)
            task.status = TaskStatus.COMPLETED
            task.result = result
            task.updated_at = datetime.now()
            
            self._notify_status_change(task)
            self.task_history.append(task)
            
            return result
            
        except Exception as e:
            task.status = TaskStatus.FAILED
            task.error = str(e)
            task.updated_at = datetime.now()
            
            self._notify_status_change(task)
            
            return f"任务执行失败: {str(e)}"
    
    async def execute_task_async(self, task_id: str) -> str:
        """异步执行任务
        
        Args:
            task_id: 任务ID
            
        Returns:
            执行结果
        """
        if task_id not in self.tasks:
            return f"任务不存在: {task_id}"
        
        task = self.tasks[task_id]
        task.status = TaskStatus.EXECUTING
        self._notify_status_change(task)
        
        try:
            result = await self.agent.execute_task_async(task.description)
            task.status = TaskStatus.COMPLETED
            task.result = result
            task.updated_at = datetime.now()
            
            self._notify_status_change(task)
            self.task_history.append(task)
            
            return result
            
        except Exception as e:
            task.status = TaskStatus.FAILED
            task.error = str(e)
            task.updated_at = datetime.now()
            
            self._notify_status_change(task)
            
            return f"任务执行失败: {str(e)}"
    
    def get_task_status(self, task_id: str) -> Optional[Dict]:
        """获取任务状态
        
        Args:
            task_id: 任务ID
            
        Returns:
            任务状态信息
        """
        if task_id in self.tasks:
            return self.tasks[task_id].to_dict()
        return None
    
    def list_tasks(self, status: TaskStatus = None) -> List[Dict]:
        """列出任务
        
        Args:
            status: 可选，按状态过滤
            
        Returns:
            任务列表
        """
        tasks = list(self.tasks.values())
        
        if status:
            tasks = [t for t in tasks if t.status == status]
        
        return [t.to_dict() for t in sorted(tasks, key=lambda x: x.created_at, reverse=True)]
    
    def cancel_task(self, task_id: str) -> bool:
        """取消任务
        
        Args:
            task_id: 任务ID
            
        Returns:
            是否成功取消
        """
        if task_id not in self.tasks:
            return False
        
        task = self.tasks[task_id]
        
        if task.status in [TaskStatus.PENDING, TaskStatus.PLANNING]:
            task.status = TaskStatus.CANCELLED
            task.updated_at = datetime.now()
            self._notify_status_change(task)
            return True
        
        return False
    
    def add_status_listener(self, listener: Callable) -> None:
        """添加状态监听器
        
        Args:
            listener: 回调函数，接收任务对象
        """
        self.status_listeners.append(listener)
    
    def _notify_status_change(self, task: Task) -> None:
        """通知状态变化
        
        Args:
            task: 状态变化的任务
        """
        for listener in self.status_listeners:
            try:
                listener(task)
            except Exception as e:
                logger.error(f"状态监听器执行失败: {str(e)}")
    
    def get_statistics(self) -> Dict:
        """获取任务统计信息
        
        Returns:
            统计信息字典
        """
        total = len(self.tasks)
        completed = len([t for t in self.tasks.values() if t.status == TaskStatus.COMPLETED])
        failed = len([t for t in self.tasks.values() if t.status == TaskStatus.FAILED])
        pending = len([t for t in self.tasks.values() if t.status == TaskStatus.PENDING])
        executing = len([t for t in self.tasks.values() if t.status == TaskStatus.EXECUTING])
        
        return {
            "total": total,
            "completed": completed,
            "failed": failed,
            "pending": pending,
            "executing": executing,
            "success_rate": round(completed / total * 100, 2) if total > 0 else 0
        }

    description: str
    task_id: Optional[str] = None
    async_mode: bool = False


class TaskResponse(BaseModel):
    """任务响应模型"""
    task_id: str
    status: str
    message: str

    """人机交互输入模型"""

@app.on_event("startup")
async def startup_event():
    """应用启动事件"""
    global agent, executor, human_interaction
    
    # 初始化组件
    agent = TaskExecutionAgent()
    executor = TaskExecutor(agent)
    human_interaction = HumanInTheLoop()
    
    # 添加状态监听器，用于WebSocket推送
    def status_listener(task: Task):
        # 广播任务状态变化
        message = json.dumps({
            "type": "status_update",
            "task": task.to_dict()
        })
        asyncio.create_task(broadcast_message(message))
    
    executor.add_status_listener(status_listener)
    
    logger.info("Agent任务执行系统启动成功")


async def broadcast_message(message: str):
    """广播消息到所有WebSocket连接"""
    disconnected = []
    
    for connection in active_connections:
        try:
            await connection.send_text(message)
        except Exception:
            disconnected.append(connection)
    
    # 移除断开的连接
    for conn in disconnected:
        if conn in active_connections:
            active_connections.remove(conn)


@app.post("/api/tasks", response_model=TaskResponse)
async def create_and_execute_task(request: TaskRequest, background_tasks: BackgroundTasks):
    """创建并执行任务
    
    Args:
        request: 任务请求
        background_tasks: 后台任务
        
    Returns:
        任务响应
    """
    task_id = request.task_id or f"task_{uuid.uuid4().hex[:8]}"
    
    # 创建任务
    task = executor.create_task(request.description, task_id)
    
    # 根据模式执行
    if request.async_mode:
        # 异步执行
        background_tasks.add_task(executor.execute_task_async, task_id)
        return TaskResponse(
            task_id=task_id,
            status="pending",
            message="任务已提交，将在后台执行"
        )
    else:
        # 同步执行
        result = executor.execute_task_sync(task_id)
        return TaskResponse(
            task_id=task_id,
            status=executor.get_task_status(task_id)["status"],
            message=result
        )


@app.get("/api/tasks/{task_id}")
async def get_task(task_id: str):
    """获取任务状态和结果
    
    Args:
        task_id: 任务ID
        
    Returns:
        任务信息
    """
    task = executor.get_task_status(task_id)
    
    if not task:
        raise HTTPException(status_code=404, detail="任务不存在")
    
    return task

@app.websocket("/ws")
async def websocket_endpoint(websocket: WebSocket):
    """WebSocket端点
    
    支持实时任务状态推送和交互
    """
    await websocket.accept()
    active_connections.append(websocket)
    
    try:
        # 发送欢迎消息
        await websocket.send_text(json.dumps({
            "type": "connected",
            "message": "WebSocket连接已建立"
        }))
        
        # 保持连接并处理消息
        while True:
            data = await websocket.receive_text()
            
            try:
                message = json.loads(data)
                msg_type = message.get("type")
                
                if msg_type == "create_task":
                    # 创建并执行任务
                    description = message.get("description")
                    task_id = f"task_{uuid.uuid4().hex[:8]}"
                    
                    # 创建任务
                    task = executor.create_task(description, task_id)
                    
                    # 异步执行
                    asyncio.create_task(executor.execute_task_async(task_id))
                    
                    await websocket.send_text(json.dumps({
                        "type": "task_created",
                        "task_id": task_id,
                        "status": "pending"
                    }))
                
                elif msg_type == "cancel_task":
                    # 取消任务
                    task_id = message.get("task_id")
                    success = executor.cancel_task(task_id)
                    
                    await websocket.send_text(json.dumps({
                        "type": "task_cancelled",
                        "task_id": task_id,
                        "success": success
                    }))
                
                elif msg_type == "human_input":
                    # 提交人类输入
                    request_id = message.get("request_id")
                    user_input = message.get("input")
                    
                    human_interaction.process_response(request_id, user_input)
                    
                    await websocket.send_text(json.dumps({
                        "type": "input_received",
                        "request_id": request_id
                    }))
                
                elif msg_type == "ping":
                    # 心跳检测
                    await websocket.send_text(json.dumps({
                        "type": "pong"
                    }))
                
                else:
                    await websocket.send_text(json.dumps({
                        "type": "error",
                        "message": f"未知消息类型: {msg_type}"
                    }))
                    
            except json.JSONDecodeError:
                await websocket.send_text(json.dumps({
                    "type": "error",
                    "message": "无效的JSON格式"
                }))
    
    except WebSocketDisconnect:
        logger.info("WebSocket客户端断开连接")
        if websocket in active_connections:
            active_connections.remove(websocket)

## 7 常见问题与解决方案

**问题一：Agent输出格式不统一**
部分模型对结构化输出的遵循程度不高，可能不按"思考/行动/完成"格式回复。解决方案是在系统提示中增加更多示例，或使用更可靠的模型（如GPT-4）。本系统采用了更健壮的解析逻辑，通过正则表达式提取不同类型的响应，同时支持"确认"类型的响应用于人机交互。

**问题二：工具参数解析错误**
当参数中包含特殊字符时，正则表达式可能无法正确解析。建议使用更健壮的参数解析方式，或要求模型输出JSON格式的参数。本系统的parse_response方法通过多层次解析，先识别响应类型，再提取具体内容，对各类情况进行了兼容处理。

**问题三：任务陷入无限循环**
Agent可能在某些情况下反复执行相同的动作。可以通过记录已执行的行动序列进行去重，或设置更小的最大迭代次数。本系统的max_iterations参数默认为10次，并在MemoryManager中实现了记忆压缩功能，避免无限循环。

**问题四：工具安全问题**
file_write、execute_code等工具存在潜在的安全风险，可能被用于写入恶意文件或执行危险代码。在生产环境中应添加权限控制和操作审计。本系统的基础实现使用了受限的执行环境，生产部署时建议使用沙箱机制。

**问题五：异步任务管理复杂性**
异步执行时，如何追踪任务状态和处理异常是一个挑战。本系统通过TaskExecutor统一管理任务状态，使用状态监听器机制实时推送任务进度，并通过WebSocket实现客户端的实时更新。

**问题六：人机交互的阻塞问题**
当Agent需要用户确认时，同步执行模式会阻塞。解决方案是使用异步模式和等待队列。本系统实现了HumanInTheLoop类，支持确认请求、选项选择、自由输入三种交互模式，并通过状态码WAITING_HUMAN标识等待确认的任务。

**问题七：大模型调用失败**
网络问题或API限制可能导致LLM调用失败。本系统集成了RetryHandler，使用指数退避策略自动重试，可配置最大重试次数和延迟时间。

**问题八：多任务并行管理**
当需要同时执行多个任务时，需要有效的任务队列和资源管理。TaskExecutor支持任务的创建、取消、状态查询，并提供统计信息功能，便于监控系统运行状态。

## 10 应用场景

**自动化工作流**：将重复性的多步骤任务（如数据处理、报表生成）交给Agent自动完成，用户只需提供任务描述即可获得结果。TaskPlanner可以将复杂任务分解为子任务序列，TaskExecutor负责调度执行。

**智能编程助手**：Agent可以帮助开发者完成代码调试、文件操作、代码审查等开发任务，提升开发效率。execute_code工具支持直接执行Python代码，file_read/file_write工具支持文件操作。

**个人助理**：完成日程安排、邮件处理、文件管理等日常办公任务，通过自然语言交互实现复杂操作。HumanInTheLoop支持关键步骤的用户确认。

**RPA增强**：作为RPA机器人的大脑，使机器人能够处理非结构化的任务描述，适应更复杂的业务流程场景。通过WebSocket实现与RPA系统的实时通信。

**数据采集与分析**：Agent可以自动搜索信息、抓取网页、分析数据，形成完整的数据处理流水线。search和fetch_url工具提供了信息获取能力。

**智能客服**：结合对话能力和工具调用，Agent可以完成订单查询、退货处理等复杂客服任务，通过记忆管理保持上下文。

**教育辅助**：Agent可以作为智能助教，回答问题、批改作业、提供个性化的学习建议，支持长期记忆来跟踪学习进度。

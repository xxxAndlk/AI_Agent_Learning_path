# 7.2 模型版本管理

## 1 技术简介

**技术定义**：模型版本管理（Model Versioning）是追踪和管理机器学习模型不同版本的过程，包括模型文件、配置文件、依赖项等，确保模型可回滚、可复现、可追溯。模型版本管理是MLOps的核心组成部分，对于生产环境中的模型部署和维护至关重要。

模型版本管理的核心价值：
- **可追溯性**：了解每个版本的训练数据、参数、指标
- **可回滚**：当新版本问题时，快速回退到之前的版本
- **可复现**：确保相同条件下可以复现相同的模型
- **合规审计**：满足监管要求，记录模型演变历史

**版本管理策略**：
- 语义版本号（Semantic Versioning）：如v1.2.3
- 时间戳版本：如20240115_v1
- 哈希版本：基于内容生成唯一标识

## 2 技术背景

**为什么需要模型版本管理**：

1. **生产环境需求**：
   - 模型需要持续迭代更新
   - 问题需要快速定位和回滚
   - 多版本并存支持A/B测试

2. **风险管理**：
   - 新模型上线可能引入问题
   - 需要保留历史版本
   - 快速恢复能力至关重要

3. **团队协作**：
   - 多人同时开发不同版本
   - 需要共享和同步模型
   - 避免版本混乱

**模型版本管理的演进**：
- 早期：手动管理，文件夹复制
- 中期：使用Git LFS存储模型
- 现代：专用模型注册表（DVC、MLflow、Weights&Biases）

## 3 核心概念

**Model Registry（模型注册表）**：
集中存储和管理模型版本的系统，提供版本追踪、状态管理、部署等功能。

**Model Version（模型版本）**：
模型的一次迭代，包含模型文件、元数据、评估指标等。

**Stage（阶段）**：
模型的生命周期阶段：
- Staging：测试环境
- Production：生产环境
- Archived：归档/弃用

**Artifact（产物）**：
模型相关文件的统称，包括模型权重、配置文件、依赖等。

**Metadata（元数据）**：
描述模型的数据，如训练数据、参数、指标、创建时间等。

## 4 工作原理

**模型版本管理系统架构**：
```
训练代码 → 模型产出 → 版本管理器 → 模型注册表 → 部署系统
                                    ↓
                              存储后端
```

**版本管理流程**：
1. 训练完成，产出模型文件
2. 记录元数据（参数、指标、数据）
3. 分配版本号
4. 上传到模型注册表
5. 可选：提升到生产环境
6. 部署时拉取指定版本

**版本命名规则**：
- 主版本号：重大架构变化
- 次版本号：新增功能
- 修订号：bug修复

## 5 技术实现

**模型注册表的基本功能**：
1. 注册新模型版本
2. 查看版本历史
3. 管理版本状态
4. 比较版本差异
5. 回滚到历史版本

**模型版本管理的最佳实践**：
1. 自动生成版本号
2. 记录完整的元数据
3. 签名和验证模型完整性
4. 设置版本生命周期策略

## 6 Python代码实现与逐行解析

```python
import os
import json
import shutil
from datetime import datetime
from typing import Dict, List, Optional

class ModelVersionManager:
    """简单模型版本管理系统
    
    功能：
    - 模型注册和版本化
    - 模型切换和回滚
    - 元数据管理
    """
    def __init__(self, registry_path="./model_registry"):
        """
        参数:
            registry_path: 模型注册表路径
        """
        self.registry_path = registry_path
        self.metadata_file = os.path.join(registry_path, "registry.json")
        
        # 初始化注册表
        os.makedirs(registry_path, exist_ok=True)
        if not os.path.exists(self.metadata_file):
            self._save_registry({
                "models": {},
                "created_at": datetime.now().isoformat()
            })
    
    def _load_registry(self) -> Dict:
        """加载注册表"""
        with open(self.metadata_file, 'r', encoding='utf-8') as f:
            return json.load(f)
    
    def _save_registry(self, registry: Dict):
        """保存注册表"""
        with open(self.metadata_file, 'w', encoding='utf-8') as f:
            json.dump(registry, f, ensure_ascii=False, indent=2)
    
    def register_model(
        self,
        model_name: str,
        model_path: str,
        version: str = None,
        description: str = "",
        tags: List[str] = None,
        metrics: Dict = None
    ) -> str:
        """注册新模型版本
        
        参数:
            model_name: 模型名称
            model_path: 模型文件路径
            version: 版本号（默认自动生成）
            description: 版本描述
            tags: 标签列表
            metrics: 评估指标
        返回:
            版本号
        """
        registry = self._load_registry()
        
        # 自动生成版本号
        if version is None:
            if model_name not in registry["models"]:
                version = "v1.0.0"
            else:
                # 简单版本递增
                last_version = registry["models"][model_name]["versions"][-1]["version"]
                major, minor, patch = last_version.replace("v", "").split(".")
                version = f"v{major}.{minor}.{int(patch)+1}"
        
        # 创建版本目录
        version_dir = os.path.join(self.registry_path, model_name, version)
        os.makedirs(version_dir, exist_ok=True)
        
        # 复制模型文件
        if os.path.isfile(model_path):
            dest_path = os.path.join(version_dir, os.path.basename(model_path))
            shutil.copy2(model_path, dest_path)
        else:
            # 如果是目录，复制整个目录
            dest_path = os.path.join(version_dir, "model")
            shutil.copytree(model_path, dest_path, dirs_exist_ok=True)
        
        # 更新注册表
        if model_name not in registry["models"]:
            registry["models"][model_name] = {
                "versions": [],
                "production_version": None
            }
        
        version_info = {
            "version": version,
            "path": version_dir,
            "description": description,
            "tags": tags or [],
            "metrics": metrics or {},
            "created_at": datetime.now().isoformat(),
            "status": "staging"  # staging/production/deprecated
        }
        
        registry["models"][model_name]["versions"].append(version_info)
        self._save_registry(registry)
        
        print(f"模型已注册: {model_name}@{version}")
        return version
    
    def promote_to_production(self, model_name: str, version: str):
        """将模型提升为生产版本"""
        registry = self._load_registry()
        
        if model_name not in registry["models"]:
            raise ValueError(f"模型不存在: {model_name}")
        
        # 查找版本
        version_info = None
        for v in registry["models"][model_name]["versions"]:
            if v["version"] == version:
                version_info = v
                break
        
        if not version_info:
            raise ValueError(f"版本不存在: {version}")
        
        # 将之前的产品版本降级
        prev_prod = registry["models"][model_name]["production_version"]
        if prev_prod:
            for v in registry["models"][model_name]["versions"]:
                if v["version"] == prev_prod:
                    v["status"] = "archived"
                    break
        
        # 提升新版本
        version_info["status"] = "production"
        registry["models"][model_name]["production_version"] = version
        
        self._save_registry(registry)
        print(f"模型 {model_name}@{version} 已提升为生产版本")
    
    def get_model(self, model_name: str, version: str = None) -> str:
        """获取模型路径
        
        参数:
            model_name: 模型名称
            version: 版本号（默认使用生产版本）
        返回:
            模型路径
        """
        registry = self._load_registry()
        
        if model_name not in registry["models"]:
            raise ValueError(f"模型不存在: {model_name}")
        
        if version is None:
            version = registry["models"][model_name].get("production_version")
            if not version:
                raise ValueError(f"模型 {model_name} 没有生产版本")
        
        for v in registry["models"][model_name]["versions"]:
            if v["version"] == version:
                return v["path"]
        
        raise ValueError(f"版本不存在: {version}")
    
    def list_models(self) -> Dict:
        """列出所有模型"""
        registry = self._load_registry()
        return registry["models"]
    
    def compare_versions(self, model_name: str, version1: str, version2: str) -> Dict:
        """比较两个版本的指标"""
        registry = self._load_registry()
        
        v1_info = None
        v2_info = None
        
        for v in registry["models"][model_name]["versions"]:
            if v["version"] == version1:
                v1_info = v
            if v["version"] == version2:
                v2_info = v
        
        if not v1_info or not v2_info:
            raise ValueError("版本不存在")
        
        return {
            version1: v1_info["metrics"],
            version2: v2_info["metrics"]
        }

if __name__ == "__main__":
    # 使用示例
    manager = ModelVersionManager()
    
    # 假设有一个模型文件
    dummy_model_path = "dummy_model.pt"
    with open(dummy_model_path, 'w') as f:
        f.write("dummy model content")
    
    # 注册模型v1
    v1 = manager.register_model(
        model_name="text_classifier",
        model_path=dummy_model_path,
        description="初始版本",
        tags=["baseline", "bert"],
        metrics={"accuracy": 0.85, "f1": 0.83}
    )
    
    # 注册模型v2
    v2 = manager.register_model(
        model_name="text_classifier",
        model_path=dummy_model_path,
        description="优化版本",
        tags=["optimized", "bert"],
        metrics={"accuracy": 0.91, "f1": 0.90}
    )
    
    # 提升v2为生产版本
    manager.promote_to_production("text_classifier", v2)
    
    # 获取生产版本路径
    prod_path = manager.get_model("text_classifier")
    print(f"\n生产版本路径: {prod_path}")
    
    # 比较版本
    comparison = manager.compare_versions("text_classifier", v1, v2)
    print(f"\n版本对比:")
    print(json.dumps(comparison, ensure_ascii=False, indent=2))
    
    # 清理
    os.remove(dummy_model_path)
```

## 7 常见问题

**模型版本管理常见问题**：

1. **模型文件过大**
   - 原因：模型权重文件体积大
   - 解决：使用Git LFS或专用存储

2. **版本混乱**
   - 原因：缺少统一的版本策略
   - 解决：制定版本命名规范

3. **回滚困难**
   - 原因：未记录完整的部署配置
   - 解决：同时版本化配置文件

4. **存储成本高**
   - 原因：保留过多历史版本
   - 解决：设置保留策略，定期清理

5. **权限管理**
   - 原因：生产环境需要权限控制
   - 解决：使用企业级模型注册表

## 8 应用场景

**模型版本管理的典型应用**：

1. **生产模型部署**
   - 稳定版本的生产发布
   - 新版本的灰度发布

2. **A/B测试**
   - 多版本并存
   - 流量分配和效果对比

3. **回滚恢复**
   - 问题版本的快速回退
   - 灾难恢复

4. **模型审计**
   - 合规要求的版本记录
   - 问题追溯

5. **实验管理**
   - 实验模型与生产模型区分
   - 快速比较不同版本

# 11.3 监控与评估

## 1 技术简介

**技术定义**：模型监控（Model Monitoring）是在生产环境中持续跟踪模型性能、数据分布、延迟等指标的系统性方法，及时发现模型退化、数据漂移等问题，确保模型持续有效。模型监控是保障AI系统可靠性的最后一道防线。

模型监控的核心价值：
- **及时发现问题**：快速检测模型性能下降
- **数据质量保障**：发现数据分布异常
- **系统健康检查**：监控推理服务状态
- **成本控制**：优化资源使用

**关键监控指标类型**：
| 指标类型 | 具体指标 | 异常表现 |
|---------|---------|---------|
| **性能指标** | 准确率、延迟、吞吐量 | 性能下降 |
| **数据漂移** | 输入分布变化、特征漂移 | 数据分布偏移 |
| **概念漂移** | 预测分布变化 | 模型过时 |
| **系统指标** | CPU/GPU使用率、内存 | 资源瓶颈 |

## 2 技术背景

**为什么需要模型监控**：

1. **生产环境的挑战**：
   - 现实世界数据不断变化
   - 模型性能可能随时间退化
   - 需要持续保障服务质量

2. **业务需求**：
   - SLA保证要求监控
   - 及时发现异常
   - 快速响应问题

3. **技术演进**：
   - 早期：无监控或简单日志
   - 中期：ELK/Grafana仪表盘
   - 现代：专用ML监控平台（Evidently AI、Fiddler）

## 3 核心概念

**数据漂移（Data Drift）**：
输入数据的分布随时间发生变化，可能导致模型性能下降。检测方法包括：
- 统计检验（KS检验、卡方检验）
- 距离度量（KL散度、JS散度）

**概念漂移（Concept Drift）**：
数据中潜在的模式发生变化，即输入与输出的关系发生改变。这比数据漂移更严重，需要重新训练模型。

**性能指标监控**：
- 延迟：P50、P95、P99响应时间
- 吞吐量：每秒请求数
- 错误率：请求失败比例

**告警（Alert）**：
当指标超过阈值时触发通知，包括：
- 邮件/短信通知
- 告警面板展示
- 自动化处理

## 4 工作原理

**监控系统架构**：
```
推理服务 → 数据收集 → 指标存储 → 告警判断 → 通知系统
                    ↓
              可视化仪表盘
```

**监控数据收集**：
1. 拦截推理请求和响应
2. 记录关键指标
3. 定期聚合统计
4. 存储到时序数据库

**告警触发逻辑**：
1. 定义告警规则（阈值、条件）
2. 持续监控指标
3. 满足条件时触发
4. 发送通知

## 5 技术实现

**监控系统的核心组件**：
1. 数据收集器：收集推理日志
2. 指标计算：聚合统计数据
3. 存储后端：时序数据库
4. 可视化：Grafana/Kibana
5. 告警系统：Prometheus Alertmanager

**监控最佳实践**：
1. 记录详细的输入输出
2. 计算有意义的指标
3. 设置合理的告警阈值
4. 建立值班响应机制

## 6 Python代码实现与逐行解析

```python
import time
import json
from datetime import datetime, timedelta
from collections import deque
from typing import Dict, List, Callable

class ModelMonitor:
    """模型监控系统
    
    功能：
    - 实时性能监控
    - 延迟追踪
    - 简单的数据漂移检测
    """
    def __init__(self, window_size: int = 1000):
        """
        参数:
            window_size: 滑动窗口大小
        """
        self.window_size = window_size
        self.metrics = {
            "latency": deque(maxlen=window_size),           # 延迟（ms）
            "throughput": deque(maxlen=window_size),        # 吞吐量（req/s）
            "errors": deque(maxlen=window_size),            # 错误数
            "predictions": deque(maxlen=window_size),       # 预测分布
            "input_stats": deque(maxlen=window_size),       # 输入统计
        }
        self.start_time = time.time()
        self.total_requests = 0
    
    def record_latency(self, latency_ms: float):
        """记录请求延迟"""
        self.metrics["latency"].append({
            "value": latency_ms,
            "timestamp": datetime.now().isoformat()
        })
    
    def record_prediction(self, prediction):
        """记录预测结果"""
        self.metrics["predictions"].append({
            "value": prediction,
            "timestamp": datetime.now().isoformat()
        })
    
    def record_error(self, error_msg: str):
        """记录错误"""
        self.metrics["errors"].append({
            "message": error_msg,
            "timestamp": datetime.now().isoformat()
        })
    
    def get_statistics(self) -> Dict:
        """获取监控统计信息"""
        stats = {
            "total_requests": self.total_requests,
            "uptime_seconds": time.time() - self.start_time,
            "error_count": len(self.metrics["errors"]),
        }
        
        # 延迟统计
        if self.metrics["latency"]:
            latencies = [m["value"] for m in self.metrics["latency"]]
            stats["latency"] = {
                "mean": sum(latencies) / len(latencies),
                "min": min(latencies),
                "max": max(latencies),
                "p95": sorted(latencies)[int(len(latencies) * 0.95)] if len(latencies) > 20 else max(latencies)
            }
        
        # 预测分布
        if self.metrics["predictions"]:
            predictions = [m["value"] for m in self.metrics["predictions"]]
            stats["prediction_distribution"] = self._calculate_distribution(predictions)
        
        return stats
    
    def _calculate_distribution(self, values: List) -> Dict:
        """计算分布统计"""
        if not values:
            return {}
        
        if isinstance(values[0], (int, float)):
            # 数值型：计算均值、方差
            mean = sum(values) / len(values)
            variance = sum((x - mean) ** 2 for x in values) / len(values)
            return {
                "mean": mean,
                "variance": variance,
                "count": len(values)
            }
        else:
            # 类别型：计算频次
            freq = {}
            for v in values:
                freq[v] = freq.get(v, 0) + 1
            return freq
    
    def check_alerts(self) -> List[str]:
        """检查告警条件"""
        alerts = []
        
        # 检查延迟
        if self.metrics["latency"]:
            latencies = [m["value"] for m in self.metrics["latency"]]
            avg_latency = sum(latencies) / len(latencies)
            if avg_latency > 1000:  # 超过1秒
                alerts.append(f"平均延迟过高: {avg_latency:.2f}ms")
        
        # 检查错误率
        if self.total_requests > 0:
            error_rate = len(self.metrics["errors"]) / self.total_requests
            if error_rate > 0.05:  # 超过5%
                alerts.append(f"错误率过高: {error_rate*100:.2f}%")
        
        return alerts

class PerformanceDecorator:
    """性能监控装饰器"""
    def __init__(self, monitor: ModelMonitor):
        self.monitor = monitor
    
    def __call__(self, func):
        """装饰器实现"""
        def wrapper(*args, **kwargs):
            start_time = time.time()
            
            try:
                result = func(*args, **kwargs)
                latency = (time.time() - start_time) * 1000  # 转为ms
                self.monitor.record_latency(latency)
                self.monitor.total_requests += 1
                
                # 如果结果是分类结果，记录预测
                if isinstance(result, (int, str)):
                    self.monitor.record_prediction(result)
                
                return result
            
            except Exception as e:
                self.monitor.record_error(str(e))
                raise
        
        return wrapper

if __name__ == "__main__":
    # 创建监控器
    monitor = ModelMonitor()
    
    # 模拟监控数据
    import random
    
    for i in range(100):
        # 模拟延迟（大部分正常，偶尔有延迟）
        if random.random() < 0.9:
            latency = random.uniform(50, 150)
        else:
            latency = random.uniform(500, 2000)
        
        monitor.record_latency(latency)
        monitor.total_requests += 1
        
        # 模拟预测
        monitor.record_prediction(random.choice(["A", "B", "C"]))
        
        # 模拟错误
        if random.random() < 0.02:
            monitor.record_error("Random error")
    
    # 获取统计
    stats = monitor.get_statistics()
    print("=== 监控统计 ===")
    print(json.dumps(stats, ensure_ascii=False, indent=2))
    
    # 检查告警
    alerts = monitor.check_alerts()
    if alerts:
        print("\n=== 告警信息 ===")
        for alert in alerts:
            print(alert)
```

## 7 常见问题

**模型监控常见问题**：

1. **监控数据量大**
   - 原因：高频推理产生大量日志
   - 解决：采样聚合，减少存储

2. **告警疲劳**
   - 原因：告警阈值设置不当
   - 解决：调优阈值，使用智能告警

3. **漂移检测困难**
   - 原因：难以定义正常范围
   - 解决：使用统计方法，设定基线

4. **跨系统追踪**
   - 原因：分布式系统调用链复杂
   - 解决：使用分布式追踪系统

5. **成本问题**
   - 原因：监控组件成本高
   - 解决：选择合适的监控方案

**模型监控的典型应用**：

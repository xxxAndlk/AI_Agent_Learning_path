# 9.1 数据清洗与预处理

## 1 技术简介

**技术定义**：数据清洗（Data Cleaning）是识别和修正数据中的错误、不一致、缺失值等问题的过程，确保数据质量。数据清洗是机器学习和AI项目中最耗时但也最重要的步骤，高质量的数据是模型性能的基础。

数据清洗的核心价值：
- **提升数据质量**：消除错误和噪声
- **保证模型性能**：防止脏数据影响模型
- **提高可解释性**：干净的数据更易分析
- **降低成本**：避免因数据问题导致的返工

**常见数据问题**：
| 问题类型 | 具体表现 | 处理方法 |
|---------|---------|---------|
| **缺失值** | 空值、NaN | 删除、填充、插值 |
| **异常值** | 离群点、错误值 | 删除、截断、变换 |
| **重复值** | 重复记录 | 删除重复 |
| **格式不一致** | 日期格式、编码 | 标准化 |
| **噪声** | 随机误差 | 平滑、滤波 |

## 2 技术背景

**为什么需要数据清洗**：

1. **现实数据的特点**：
   - 采集过程可能引入错误
   - 多来源数据格式不统一
   - 存在各种数据质量问题

2. **对模型的影响**：
   - 脏数据导致模型性能下降
   - 缺失值影响训练
   - 异常值干扰模型学习

3. **行业实践**：
   - 数据科学家80%时间用于清洗
   - 清洗效果直接影响项目成功

## 3 核心概念

**缺失值（Missing Value）**：
数据中为空的值，可能原因：
- 数据未采集
- 采集错误
- 存储问题

**异常值（Outlier）**：
偏离正常范围的数据点，可能是：
- 真实的极端情况
- 数据录入错误
- 测量误差

**重复值（Duplicate）**：
完全相同或基本相同的记录，会导致：
- 模型过拟合
- 评价指标不准确

**数据标准化**：
将数据转换为统一格式，包括：
- 日期格式统一
- 编码格式统一
- 文本大小写统一

## 4 工作原理

**数据清洗流程**：
```
数据加载 → 问题识别 → 数据处理 → 质量验证 → 输出
```

**缺失值处理策略**：
1. 删除：适用于缺失比例低
2. 填充：均值、中位数、众数
3. 插值：适用于时序数据
4. 预测：用模型预测缺失值

**异常值检测方法**：
1. IQR方法：基于四分位距
2. Z-score方法：基于标准差
3. 箱线图：可视化异常值

**文本标准化**：
- 小写转换
- 去除标点
- 去除多余空格

## 5 技术实现

**清洗策略选择原则**：
1. 了解数据来源和问题
2. 分析缺失/异常比例
3. 选择合适的处理方法
4. 验证处理效果

**最佳实践**：
1. 先备份原始数据
2. 记录所有清洗步骤
3. 验证清洗后的质量
4. 自动化可重复流程

## 6 Python代码实现与逐行解析

```python
import pandas as pd                   # 数据处理库
import numpy as np                    # 数值计算
from typing import List, Union, Optional

class DataCleaner:
    """数据清洗工具类"""
    
    @staticmethod
    def handle_missing_values(
        df: pd.DataFrame,
        strategy: str = "mean",
        fill_value=None
    ) -> pd.DataFrame:
        """处理缺失值
        
        参数:
            df: 输入数据框
            strategy: 填充策略 (mean/median/mode/constant/drop)
            fill_value: 当strategy='constant'时的填充值
        返回:
            处理后的数据框
        """
        df_clean = df.copy()
        
        # 显示缺失值统计
        missing_stats = df_clean.isnull().sum()
        missing_stats = missing_stats[missing_stats > 0]
        if len(missing_stats) > 0:
            print("缺失值统计:")
            print(missing_stats)
        
        if strategy == "drop":
            # 删除包含缺失值的行
            df_clean = df_clean.dropna()
        elif strategy == "mean":
            # 用均值填充数值列
            for col in df_clean.select_dtypes(include=[np.number]).columns:
                df_clean[col].fillna(df_clean[col].mean(), inplace=True)
        elif strategy == "median":
            # 用中位数填充数值列
            for col in df_clean.select_dtypes(include=[np.number]).columns:
                df_clean[col].fillna(df_clean[col].median(), inplace=True)
        elif strategy == "mode":
            # 用众数填充
            for col in df_clean.columns:
                if not df_clean[col].mode().empty:
                    df_clean[col].fillna(df_clean[col].mode()[0], inplace=True)
        elif strategy == "constant":
            # 用固定值填充
            df_clean.fillna(fill_value, inplace=True)
        elif strategy == "ffill":
            # 前向填充
            df_clean.fillna(method='ffill', inplace=True)
        elif strategy == "bfill":
            # 后向填充
            df_clean.fillna(method='bfill', inplace=True)
        
        return df_clean
    
    @staticmethod
    def remove_outliers(
        df: pd.DataFrame,
        columns: List[str],
        method: str = "iqr",
        threshold: float = 1.5
    ) -> pd.DataFrame:
        """删除异常值
        
        参数:
            df: 输入数据框
            columns: 要处理的列
            method: 检测方法 (iqr/zscore)
            threshold: 阈值
        返回:
            处理后的数据框
        """
        df_clean = df.copy()
        
        for col in columns:
            if col not in df_clean.columns:
                continue
            
            if method == "iqr":
                # IQR方法
                Q1 = df_clean[col].quantile(0.25)
                Q3 = df_clean[col].quantile(0.75)
                IQR = Q3 - Q1
                lower_bound = Q1 - threshold * IQR
                upper_bound = Q3 + threshold * IQR
                
                # 标记异常值
                outliers = (df_clean[col] < lower_bound) | (df_clean[col] > upper_bound)
                print(f"{col}: 发现 {outliers.sum()} 个异常值")
                
                # 删除异常值
                df_clean = df_clean[~outliers]
            
            elif method == "zscore":
                # Z-score方法
                z_scores = np.abs((df_clean[col] - df_clean[col].mean()) / df_clean[col].std())
                outliers = z_scores > threshold
                print(f"{col}: 发现 {outliers.sum()} 个异常值")
                df_clean = df_clean[~outliers]
        
        return df_clean
    
    @staticmethod
    def remove_duplicates(
        df: pd.DataFrame,
        subset: Optional[List[str]] = None,
        keep: str = "first"
    ) -> pd.DataFrame:
        """删除重复值
        
        参数:
            df: 输入数据框
            subset: 仅考虑特定列
            keep: 保留哪个 (first/last/False)
        返回:
            处理后的数据框
        """
        duplicates = df.duplicated(subset=subset).sum()
        print(f"发现 {duplicates} 个重复记录")
        
        return df.drop_duplicates(subset=subset, keep=keep)
    
    @staticmethod
    def standardize_text(
        df: pd.DataFrame,
        columns: List[str],
        lowercase: bool = True,
        remove_punctuation: bool = False,
        remove_whitespace: bool = True
    ) -> pd.DataFrame:
        """标准化文本
        
        参数:
            df: 输入数据框
            columns: 文本列
            lowercase: 转为小写
            remove_punctuation: 删除标点
            remove_whitespace: 删除多余空白
        返回:
            处理后的数据框
        """
        df_clean = df.copy()
        import string
        
        for col in columns:
            if col not in df_clean.columns:
                continue
            
            # 转为字符串类型
            df_clean[col] = df_clean[col].astype(str)
            
            if lowercase:
                df_clean[col] = df_clean[col].str.lower()
            
            if remove_punctuation:
                df_clean[col] = df_clean[col].apply(
                    lambda x: x.translate(str.maketrans('', '', string.punctuation))
                )
            
            if remove_whitespace:
                df_clean[col] = df_clean[col].str.replace(r'\s+', ' ', regex=True).str.strip()
        
        return df_clean

if __name__ == "__main__":
    # 创建示例数据
    np.random.seed(42)
    data = {
        'A': [1, 2, np.nan, 4, 5, 100, 7, 8],           # 包含缺失值和异常值
        'B': [10, 20, 30, 40, 50, 60, 70, np.nan],      # 包含缺失值
        'C': ['Hello', 'World', 'hello', 'WORLD', None, '  Test  ', 'Test', 'test'],
    }
    df = pd.DataFrame(data)
    
    print("原始数据:")
    print(df)
    print()
    
    # 创建清洗器
    cleaner = DataCleaner()
    
    # 1. 处理缺失值
    df_clean = cleaner.handle_missing_values(df, strategy="mean")
    print("\n处理缺失值后:")
    print(df_clean)
    
    # 2. 处理异常值
    df_clean = cleaner.remove_outliers(df_clean, columns=['A'], method="iqr")
    print("\n处理异常值后:")
    print(df_clean)
    
    # 3. 标准化文本
    df_clean = cleaner.standardize_text(
        df_clean,
        columns=['C'],
        lowercase=True,
        remove_whitespace=True
    )
    print("\n标准化文本后:")
    print(df_clean)
```

## 7 常见问题

**数据清洗常见问题**：

1. **过度清洗**
   - 原因：删除过多有效数据
   - 解决：评估清洗影响

2. **填充不當**
   - 原因：使用不合适的填充值
   - 解决：了解数据分布

3. **信息丢失**
   - 原因：异常值处理过于激进
   - 解决：保留异常值记录

4. **清洗顺序**
   - 原因：顺序不当影响效果
   - 解决：制定标准流程

5. **可重复性**
   - 原因：手动清洗无法复现
   - 解决：使用代码自动化

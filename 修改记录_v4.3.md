## 一、LangChain框架全面更新至v1.x

### 1. 版本信息校正
**问题**：v4.2中LangChain版本描述存在虚构内容（Deep Agents、Guardrails、各种虚构Middleware类），且Agent API已严重过时。

**修复**：
- 删除所有虚构特性：`Deep Agents`、`Guardrails作为内置特性`、`SummarizationMiddleware`、`HumanInTheLoopMiddleware`、`RateLimitMiddleware`、`PIIDetectionMiddleware`、`ModelRetryMiddleware`、`ToolRetryMiddleware`、`ShellToolMiddleware`
- 修正版本号：`langchain-core` 1.2.31+ / 1.3.0alpha，`langchain-classic` 1.0.4
- Python要求：3.9 → 3.10+

### 2. Agent API迁移
**旧API（已废弃）**：
- `from langgraph.prebuilt import create_react_agent`
- `from langchain.agents import AgentExecutor, create_openai_functions_agent, initialize_agent`

**新API（v1.x推荐）**：
- `from langchain.agents import create_agent`
- 参数 `prompt` → `system_prompt`
- Middleware架构：`before_model`, `after_model`, `wrap_tool_call`
- 动态提示：`@dynamic_prompt` decorator
- 结构化输出策略：`ToolStrategy` / `ProviderStrategy`

**涉及文件**：
- `4.1_LangChain基础与架构.md`
- `4.2_LCEL表达式详解.md`
- `4.3_Memory组件详解.md`
- `4.4_Tools与Agent集成.md`

## 二、LLM模型全面更新至GPT-5.4系列

### 背景
OpenAI已于2026年2月退役gpt-4o系列API，`chatgpt-4o-latest`等快照已停止服务。当前最新模型为GPT-5.4系列。

### 模型名称替换
| 旧参考（已退役/过时） | 新参考 | 说明 |
|---------|---------|------|
| `gpt-4o-mini` | `gpt-5.4-mini` | 高性价比模型，支持tool search、computer use、compaction |
| `gpt-4o` | `gpt-5.4` | 主力多模态模型，1M token上下文 |
| `gpt-4` | `gpt-5.4` | 前沿专业模型 |
| `gpt-3.5-turbo` | `gpt-5.4-nano` | 简单高容量任务，速度最快 |
| `claude-3-opus` | `claude-4-opus` | Anthropic最新旗舰 |
| `claude-3` 系列 | `claude-4` 系列 | Anthropic新一代模型 |

### 涉及文件
共计36个Markdown文件中的模型名称得到更新，覆盖全部14章。
